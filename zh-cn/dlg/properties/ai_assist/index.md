# “AI 辅助”页面

**AI 辅助**让你能设置与 AI 辅助相关的属性。

## “启用 AI 辅助写作”复选框

如果选中此项，EmEditor 将启用 AI 辅助写作。

## “仅在按下 Ctrl+Space 时显示建议”复选框

如果勾选此项，EmEditor 仅会在你按下快捷键 Ctrl+Space 时显示建议，这有助于减少 AI 调用的频率。

## “仅在有把握时才显示建议”复选框

如果选中此项，则会启用“置信水平”滑块。

## “设置温度”复选框

在 AI 中，**温度**是一个参数，用于控制大语言模型在生成文本、选择下一个词时所使用的“随机性（创造力）”。温度越低，随机性越低，你收到的建议就越确定、越具有可预测性。温度越高，随机性越高，你收到的建议就越有创造力。当**设置温度**选项关闭时，始终使用模型的默认温度。当该选项开启时，初始会使用 0.0 的温度，以便你获得确定性的建议；不过，当你按下 Ctrl + Space 请求不同的建议时，温度会提高，从而生成更具创造性的建议。许多近期的推理模型不接受温度参数，此时该选项会被忽略。如果由于某个模型无法设置温度而导致状态栏出现错误，请关闭此选项。对于 AI 辅助写作，我们建议指定一个能够快速返回响应的轻量级模型，而不是推理模型。

## “在以下任一字符之后不显示建议”复选框

如果勾选此项，并在下方字段中输入表示句子结束的字符（例如“.。”），则在“.”或“。”等符号之后将不再显示建议。这样，你可以根据正在编辑的文本类型，按需调整此功能以便于使用。

## “置信水平”滑块

使用此滑块调整 AI 决策的置信水平。例如，如果将置信设置为 40%，AI 将仅在预测下一个文本时至少有 40% 的置信水平时才显示建议。要减少不太确定预测的建议，可以设置更高的置信水平，例如 80%，确保仅在预测更可靠时才显示建议，从而减少 OpenAI API 调用的次数。

## “输入长度”滑块

使用此滑块调整输入长度。使用较长的输入文本进行预测可以提高准确性，但也会增加 OpenAI API 的使用费用。

## “输出长度”滑块

使用此滑块调整输出长度。较长的输出文本会建议更多的单词。

## “延迟时间”滑块

当停止输入且编辑器进入空闲状态后，经过指定的时间间隔，AI 建议会自动显示。你可以使用滑块以毫秒为单位调整此延迟时间。

## “额外指令”文本框

在这里，你可以添加一些指示，例如希望收到什么类型的建议、使用哪种语言，以及期望的语气。例如，在用 JavaScript 编写代码时，你可以指定“使用 JavaScript”；如果你希望使用礼貌用语，也可以指定“以礼貌的风格撰写”。不过，根据 AI 模型的不同，这些附加指示不一定总能被完全理解或生成恰当的回应，而且输入过多指示可能会降低响应速度。

## 「重置」按钮

重置为默认设定。会显示[**重置**对话框](../reset/index)并让你能从另一个配置复制设定。

## 注意

仅当安装并启用了 **ChatAI** 插件时，此页面才可用。
