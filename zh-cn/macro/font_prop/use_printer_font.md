# UsePrinterFont 属性

与自定义字体对话框中的 [**在下方的列表中包含默认的打印机字体** 复选框](../../dlg/properties/font/index) 相对应。

#### \[JavaScript\]

_b_ =
object. **UsePrinterFont**;

object. **UsePrinterFont** = _b_;

#### \[VBScript\]

_b_ =
object. **UsePrinterFont**

object. **UsePrinterFont** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
