# writeln 方法 (Interface 对象)

把一个字符串和新行传递到一个支持的插件中。

## 

### \[JavaScript\]

```
Interface.writeln( str );
```

### \[VBScript\]

```
Interface.writeln str
```

## 参数

_str_

指定要传递的字符串。

## 示例

### \[JavaScript\]

```
Interface.writeln( "Hello" );
```

### \[VBScript\]

```
Interface.writeln "Hello"
```

## 版本

支持 EmEditor 9.00 或之后的版本。
