# RearrangeColumns 方法 (Document )

重排 CSV 列。

#### \[JavaScript\]

document. **RearrangeColumns**( _strColumns_ );

#### \[VBScript\]

document. **RearrangeColumns** _strColumns_

## 参数

_strColumns_

> 指定一个字符串来指示要重新排列的列的顺序。例如，"1, 3, 5" 表示结果将包括原始 CSV 文档的第一列，第三列，和第五列。

## 版本

支持 EmEditor Professional v22.1 或之后的版本。
