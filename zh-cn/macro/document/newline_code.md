# NewlineCode 属性 (Document 对象)

检索文档的当前换行字符码。

## 

### \[JavaScript\]

```
nNewlineCode = document.NewlineCode;
```

### \[VBScript\]

```
nNewlineCode = document.NewlineCode
```

## 参数

_nNewlineCode_

指定下列其中一个值。

|     |     |
| --- | --- |
| eeCrAndLf | CR+LF |
| eeCrOnly | 仅 CR |
| eeLfOnly | 仅 LF |
| eeNewlineMixed | 混合 |

## 版本

支持 EmEditor Professional 17.6 或之后的版本。
