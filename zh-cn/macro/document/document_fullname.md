# FullName 属性 (Document 对象)

检索文档的完整路径以及文件名称。

## 

### \[JavaScript\]

```
strName = document.FullName;
```

### \[VBScript\]

```
strName = document.FullName
```

## 示例

### \[JavaScript\]

```
alert( document.FullName );
```

### \[VBScript\]

```
alert document.FullName
```

## 版本

支持 EmEditor 4.00 或之后的版本。
