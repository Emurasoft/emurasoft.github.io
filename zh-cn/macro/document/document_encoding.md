# Encoding 属性 (Document 对象)

检索或设置打开的文件的当前编码。

## 

### \[JavaScript\]

```
nEncoding = document.Encoding;
document.Encoding = nEncoding;
```

### \[VBScript\]

```
nEncoding = document.Encoding
document.Encoding = nEncoding
```

## 参数

_nEncoding_

从 **[编码常数](../const/const_encoding)** 中选择或指定任何用于 Windows 操作系统的代码页。

## 版本

支持 EmEditor 4.00 或之后的版本。
