# FontCategory 属性 (Document 对象)

检索或设置当前字体类别。

## 

### \[JavaScript\]

```
nCategory = document.FontCategory;
document.FontCategory = nCategory;
```

### \[VBScript\]

```
nCategory = document.FontCategory
document.FontCategory = nCategory
```

## 参数

_nCategory_

从下列值中选择。

|     |     |
| --- | --- |
| eeFontCategoryDefault | 标准 |
| eeFontCategoryArabic | 阿拉伯文 |
| eeFontCategoryBaltic | 波罗的海文 |
| eeFontCategoryCentralEuropean | 中欧文 |
| eeFontCategoryChineseSimplified | 简体中文 |
| eeFontCategoryChineseTraditional | 繁体中文 |
| eeFontCategoryCyrillic | 西里尔文 |
| eeFontCategoryGreek | 希腊文 |
| eeFontCategoryHebrew | 希伯来文 |
| eeFontCategoryJapanese | 日文 |
| eeFontCategoryKorean | 韩文 |
| eeFontCategoryThai | 泰文 |
| eeFontCategoryTurkish | 土耳其文 |
| eeFontCategoryVietnamese | 越南文 |
| eeFontCategoryWesternEuropean | 西欧文 |
| eeFontCategoryOEM | OEM/DOS |

## 版本

支持 EmEditor 4.00 或之后的版本。
