# Windows Scripting Host ()

宏的操作基于 Windows Scripting Host (WSH) 引擎的工作环境。因此，你可以用不同的Windows Component 对象 Model (COM) 对象。例如你可以用 RegExp 对象来执行正则表达式搜索，用 FileSystemObject 对象来操控文件，用 WshShell 对象创建快捷键，操纵 Windows 注册表，或者用 WshNetwork 对象来进行具有联网功能的工作。另外，你可以创建并执行宏，让你能用一个支持自动化（例如 Word 和 Excel）的外部程序来复制 EmEditor 中创建的文档，粘贴并显示在外部程序中。
