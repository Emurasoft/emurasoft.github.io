# IgnoreAfter 属性 (IndentProp 对象)

与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的忽略正则表达式之后的内容 文本框相对应。

## 

### \[JavaScript\]

```
s = object.IgnoreAfter;
object.IgnoreAfter = s;
```

### \[VBScript\]

```
s = object.IgnoreAfter
object.IgnoreAfter = s
```

## 版本

支持 EmEditor 9.00 或之后的版本。
