# TabColumns 属性 (IndentProp 对象)

与 [**制表符/缩进** 对话框](../../dlg/properties/general/indent/index) 上的 **Tab 列** 文本框相对应。

## 

### \[JavaScript\]

```
n = object.TabColumns;
object.TabColumns = n;
```

### \[VBScript\]

```
n = object.TabColumns
object.TabColumns = n
```

## 版本

支持 EmEditor 7.00 或之后的版本。
