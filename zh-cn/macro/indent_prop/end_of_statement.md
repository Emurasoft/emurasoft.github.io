# EndOfStatement 属性 (IndentProp 对象)

与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的语句结束 文本框相对应。

## 

### \[JavaScript\]

```
s = object.EndOfStatement;
object.EndOfStatement = s;
```

### \[VBScript\]

```
s = object.EndOfStatement
object.EndOfStatement = s
```

## 版本

支持 EmEditor 7.00 或之后的版本。
