# SelectColumn 方法 (Selection 对象)

在 CSV 模式中选择指定的列。

## 

### \[JavaScript\]

```
document.selection.SelectColumn( iColumn, [ [ iColumn2 ], nFlags ] );
```

### \[VBScript\]

```
document.selection.SelectColumn iColumn, [ [ iColumn2 ], nFlags ]
```

## 参数

_nFlags_

指定 0 或下列值：

|     |     |
| --- | --- |
| eeNoHeadings | 选取内容排除标题。 |

## 返回值

此方法不返回值。

## 版本

支持 EmEditor Professional 21.0 或之后的版本。
