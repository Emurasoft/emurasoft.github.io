# LineOpen 方法 (Selection )

在两行之间插入一个空行。

#### \[JavaScript\]

document.selection. **LineOpen**( \[ _bAbove_ \] );

#### \[VBScript\]

document.selection. **LineOpen** \[ _bAbove_ \]

## 参数

_bAbove_

可选项。如果 true 被指定，在当前行的开始插入一个新行。如果 false 被指定，在当前行的末尾插入一个新行。默认值是 false。

## 版本

支持 EmEditor 4.00 或之后的版本。
