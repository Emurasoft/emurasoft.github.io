# FindRepeat 方法

对同一个字符串重复上一次搜索。

#### \[JavaScript\]

document.selection. **FindRepeat**( _nFlags_ );

#### \[VBScript\]

document.selection. **FindRepeat** _nFlags_

## 参数

_nFlags_

指定一个下列值的组合:

|     |     |
| --- | --- |
| eeFindRepeatNext | 从光标处往下搜索字符串。 |
| eeFindRepeatPrevious | 从光标处往上搜索字符串。 |
| eeFindRepeatWord | 搜索光标处的单词如果选定内容为空的话。 |

## 版本

支持 EmEditor 4.00 或之后的版本。
