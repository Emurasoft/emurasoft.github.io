# StartOfDocument 方法 (Selection 对象)

把光标移动到文档的开始位置。

## 

### \[JavaScript\]

```
document.selection.StartOfDocument( [ bExtend ] );
```

### \[VBScript\]

```
document.selection.StartOfDocument [ bExtend ]
```

## 参数

_bExtend_

可选项。指定是否折叠被移动的文本。默认的值为 false，并且所移动的文本被折叠。

## 版本

支持 EmEditor 4.00 或之后的版本。
