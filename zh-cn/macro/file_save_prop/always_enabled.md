# AlwaysEnabled 属性 (FileSaveProp 对象)

与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的总是启用保存 复选框相对应。

## 

### \[JavaScript\]

```
b = object.AlwaysEnabled;
object.AlwaysEnabled = b;
```

### \[VBScript\]

```
b = object.AlwaysEnabled
object.AlwaysEnabled = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
