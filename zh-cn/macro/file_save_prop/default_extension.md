# DefaultExtension 属性 (FileSaveProp 对象)

与配置属性中 [文件 页面](../../dlg/properties/file/index) 上的默认扩展名 文本框相对应。

## 

### \[JavaScript\]

```
s = object.DefaultExtension;
object.DefaultExtension = s;
```

### \[VBScript\]

```
s = object.DefaultExtension
object.DefaultExtension = s
```

## 版本

支持 EmEditor 7.00 或之后的版本。
