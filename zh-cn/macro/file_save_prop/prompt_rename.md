# PromptRename 属性 (FileSaveProp 对象)

与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的更名时在删除旧文件前提示 复选框相对应。

## 

### \[JavaScript\]

```
b = object.PromptRename;
object.PromptRename = b;
```

### \[VBScript\]

```
b = object.PromptRename
object.PromptRename = b
```

## 版本

支持 EmEditor 13 或之后的版本。
