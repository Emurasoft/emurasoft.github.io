# PromptRename 属性

与 [**保存详细信息** 对话框](../../dlg/properties/file/save_details/index) 中的 **更名时在删除旧文件前提示** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **PromptRename**;

object. **PromptRename** = _b_;

#### \[VBScript\]

_b_ =
object. **PromptRename**

object. **PromptRename** = _b_

## 版本

支持 EmEditor 13 或之后的版本。
