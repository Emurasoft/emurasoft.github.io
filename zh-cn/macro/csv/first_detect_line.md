# FirstDetectLine 属性

与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **要检测的首行** 文本框对应。

#### \[JavaScript\]

_n_ = object. **FirstDetectLine**;

object. **FirstDetectLine** = _n_;

#### \[VBScript\]

_n_ = object. **FirstDetectLine**

object. **FirstDetectLine** = _n_

## 版本

支持 EmEditor 19.4 或之后的版本。
