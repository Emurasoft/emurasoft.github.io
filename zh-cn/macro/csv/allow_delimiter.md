# AllowDelimiter 属性 (Csv 对象)

与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **允许分隔符在引号内** 复选框对应。

## 

### \[JavaScript\]

```
b = object.AllowDelimiter;
object.AllowDelimiter = b;
```

### \[VBScript\]

```
b = object.AllowDelimiter
object.AllowDelimiter = b
```

## 版本

支持 EmEditor 19.4 或之后的版本。
