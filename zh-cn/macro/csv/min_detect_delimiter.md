# MinDetectDelimiter 属性 (Csv 对象)

与在自定义对话框中 [CSV 页面](../../dlg/customize/csv/index) 上的要检测的最少分隔符数 文本框对应。

## 

### \[JavaScript\]

```
n = object.MinDetectDelimiter;
object.MinDetectDelimiter = n;
```

### \[VBScript\]

```
n = object.MinDetectDelimiter
object.MinDetectDelimiter = n
```

## 版本

支持 EmEditor 19.4 或之后的版本。
