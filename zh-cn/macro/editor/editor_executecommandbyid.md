# ExecuteCommandByID 方法 (Editor )

执行由一个表示命令 ID 整数标识的命令。

## 

### \[JavaScript\]

```
editor.ExecuteCommandByID( nID );
```

### \[VBScript\]

```
strName = editor.ExecuteCommandByID nID
```

## 参数

_nID_

指定一个表示要执行的命令 ID 的整数。详见
[命令参考](../../cmd/index) 来查看可用命令列表。不是所有的命令都可用或被支持。

## 版本

支持 EmEditor 4.00 或之后的版本。
