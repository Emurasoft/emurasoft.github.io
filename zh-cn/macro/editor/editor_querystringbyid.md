# QueryStringByID 方法 (Editor )

检索与指定命令相关联的字符串。

#### \[JavaScript\]

_str_ = editor. **QueryStringByID**( _nID_ );

#### \[VBScript\]

_str_ = editor. **QueryStringByID**( _nID_ )

## 参数

_nID_

指定表示要执行的命令 ID 的一个整数。只能用下列命令。详见 **[命令参考](../../cmd/index)**。

| --- | --- | --- |
| nID | 命令名称 | 返回值 |
| 4609 到 4609 + 63 | [**最近使用的文档列表**](../../cmd/file/file_mru_file1) | 文件路径和名称 |

| 5376 到 5376 + 255 | **[文档列表](../../cmd/window/window_menu)** | 窗口标题 |

## 返回值

返回字符串，该字符串取决于 nID 参数的含义。如果 nID 参数无效，该方法会返回一个空字符串。

## 版本

支持 EmEditor 7.00 或之后的版本。
