# UseRecycleBin 属性 (BackupProp 对象)

与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的使用回收站进行备份 复选框对应。

## 

### \[JavaScript\]

```
b = object.UseRecycleBin;
object.UseRecycleBin = b;
```

### \[VBScript\]

```
b = object.UseRecycleBin
object.UseRecycleBin = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
