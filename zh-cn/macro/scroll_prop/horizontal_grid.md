# HorizontalGrid 属性

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **水平格线** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **HorizontalGrid**;

object. **HorizontalGrid** = _b_;

#### \[VBScript\]

_b_ =
object. **HorizontalGrid**

object. **HorizontalGrid** = _b_

## 版本

支持 EmEditor 10.00 或之后的版本。
