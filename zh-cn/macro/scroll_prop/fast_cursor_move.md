# FastCursorMove 属性

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **快速移动光标** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **FastCursorMove**;

object. **FastCursorMove** = _b_;

#### \[VBScript\]

_b_ =
object. **FastCursorMove**

object. **FastCursorMove** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
