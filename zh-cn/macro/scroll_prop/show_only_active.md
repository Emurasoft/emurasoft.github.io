# ShowOnlyActive 属性 (ScrollProp 对象)

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **只在活动窗格中显示滚动条** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.ShowOnlyActive;
object.ShowOnlyActive = b;
```

### \[VBScript\]

```
b = object.ShowOnlyActive
object.ShowOnlyActive = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
