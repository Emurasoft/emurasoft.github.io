# HorizontalBar 属性 (ScrollProp 对象)

与配置属性中 [滚动 页面](../../dlg/properties/scroll/index) 上的水平滚动条 下拉列表框相对应。

## 

### \[JavaScript\]

```
n = object.HorizontalBar;
object.HorizontalBar = n;
```

### \[VBScript\]

```
n = object.HorizontalBar
object.HorizontalBar = n
```

## 版本

支持 EmEditor 7.00 或之后的版本。
