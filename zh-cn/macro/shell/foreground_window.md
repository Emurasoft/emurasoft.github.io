# ForegroundWindow 属性 (Shell 对象)

检索当前置前窗口。

## 

### \[JavaScript\]

```
wnd = shell.ForegroundWindow;
```

### \[VBScript\]

```
wnd = shell.ForegroundWindow
```

## 示例

### \[JavaScript\]

```
wnd = shell.FouregroundWindow;
str = wnd.Caption;
alert( str );
```

### \[VBScript\]

```
wnd = shell.FouregroundWindow
str = wnd.Caption
alert str
```

## 版本

支持 EmEditor 7.00 或之后的版本。
