# confirm 方法

在一个简单的对话框内显示一条消息以及「OK」和「Cancel」按钮。

#### \[JavaScript\]

_bAnswer_ = **confirm**( _strQuestion, strMessage2_);

#### \[VBScript\]

_bAnswer_ = **confirm**( _strQuestion, strMessage2_ )

## 参数

_strQuestion_

指定要显示的消息。

_strMessage2_

可选项。指定要在第一条消息下方显示的额外的消息。

## 返回值

返回 true 如果选择「OK」按钮，或者返回 false 如果选择「Cancel」按钮。

## 版本

支持 EmEditor 4.00 或之后的版本。
