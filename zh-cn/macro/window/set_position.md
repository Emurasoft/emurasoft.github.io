# SetPosition 方法 (Window )

设置窗口的大小和位置。

#### \[JavaScript\]

wnd. **SetPosition**( _x_, _y_, _width_, _height_ );

#### \[VBScript\]

wnd. **SetPosition** _x_, _y_, _width_, _height_

## 参数

_x_

以像素为单位指定窗口的水平位置。

_y_

以像素为单位指定窗口的垂直位置。

_width_

以像素为单位指定窗口的宽度。

_height_

以像素为单位指定窗口的高度。

## 版本

支持 EmEditor 7.00 或之后的版本。
