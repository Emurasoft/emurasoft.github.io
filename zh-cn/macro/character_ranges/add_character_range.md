# AddCharacterRange 方法 (CharacterRanges 集合)

添加字符范围。

## 

### \[JavaScript\]

```
list.AddCharacterRange( nMin, nMax, sProperties, sTreatAs );
```

### \[VBScript\]

```
list.AddCharacterRange nMin, nMax, sProperties, sTreatAs
```

## 参数

_nMin_

指定范围的最小字符。

_nMax_

指定范围的最大字符。

_sProperties_

指定表示 Unicode 脚本和/或 Unicode 一般类别属性的字符串。

_sTreatAs_

指定要替换的字符串。如果指定了空字符串，则指定的范围将被忽略。

## 版本

支持 EmEditor Professional Version 22.0 或之后的版本。
