# SaveToFolder 属性

与配置属性中 [**自动保存** 页面](../../dlg/properties/autosave/index) 上的 **保存到自动保存文件夹** 复选框相对应。

#### \[JavaScript\]

_b_ = object. **SaveToFolder**;

object. **SaveToFolder** = _b_;

#### \[VBScript\]

_b_ = object. **SaveToFolder**

object. **SaveToFolder** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
