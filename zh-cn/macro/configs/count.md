# Count 属性 (Configs 集合)

检索配置的数目。

## 

### \[JavaScript\]

```
nCount = editor.Configs.Count;
```

### \[VBScript\]

```
nCount = editor.Configs.Count
```

## 示例

### \[JavaScript\]

```
alert( "Number of Configurations: " + editor.Configs.Count );
```

### \[VBScript\]

```
alert "Number of Configurations: " & editor.Configs.Count
```

## 版本

支持 EmEditor 7.00 或之后的版本。
