# SingleQuotes 属性 (SpellProp 对象)

与配置属性中 [**拼写检查** 页面](../../dlg/properties/spell/index) 上的 **单引号内的字符串** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.SingleQuotes;
object.SingleQuotes = b;
```

### \[VBScript\]

```
b = object.SingleQuotes
object.SingleQuotes = b
```

## 版本

支持 EmEditor 10.00 或之后的版本。
