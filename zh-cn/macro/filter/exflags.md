# ExFlags 属性 (Filter 对象)

指定下列值的组合。

|     |     |
| --- | --- |
| eeExFindNumberRange | 匹配 [数字范围表达式](../../howto/search/number_range_syntax)。此标志不能与 eeFindReplaceEscSeq 或 eeFindReplaceRegExp 合用。 |

## 

### \[JavaScript\]

```
flag = item.ExFlags;
item.ExFlags = flags;
```

### \[VBScript\]

```
n = item.ExFlags
item.ExFlags = n
```

## 版本

支持 EmEditor Professional 19.9 或之后的版本。
