# NoFullPathIfNotActive 属性 (FileProp 对象)

与配置属性中 [文件 页面](../../dlg/properties/file/index) 上的失去焦点时不显示完整路径 复选框相对应。

## 

### \[JavaScript\]

```
b = object.NoFullPathIfNotActiveProperty;
object.NoFullPathIfNotActiveProperty = b;
```

### \[VBScript\]

```
b = object.NoFullPathIfNotActiveProperty
object.NoFullPathIfNotActiveProperty = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
