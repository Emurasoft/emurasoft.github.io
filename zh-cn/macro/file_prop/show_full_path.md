# ShowFullPath 属性 (FileProp )

与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **显示包含完整路径的文件名** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **ShowFullPath**;

object. **ShowFullPath** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowFullPath**

object. **ShowFullPath** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
