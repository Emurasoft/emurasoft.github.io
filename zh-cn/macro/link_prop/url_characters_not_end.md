# URLCharactersNotEnd 属性

与配置属性中 [**链接** 页面](../../dlg/properties/link/index) 上的 **行末不允许的字符** 复选框相对应。

#### \[JavaScript\]

_s_ = object. **URLCharactersNotEnd**;

object. **URLCharactersNotEnd** = _s_;

#### \[VBScript\]

_s_ = object. **URLCharactersNotEnd**

object. **URLCharactersNotEnd** = _s_

## 版本

支持 EmEditor 7.00 或之后的版本。
