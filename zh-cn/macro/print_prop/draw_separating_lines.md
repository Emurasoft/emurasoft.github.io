# DrawSeparatingLines 属性 (PrintProp 对象)

与配置属性中 [打印 页面](../../dlg/properties/print/index) 上的插入分隔线 复选框相对应。

## 

### \[JavaScript\]

```
b = object.DrawSeparatingLines;
object.DrawSeparatingLines = b;
```

### \[VBScript\]

```
b = object.DrawSeparatingLines
object.DrawSeparatingLines = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
