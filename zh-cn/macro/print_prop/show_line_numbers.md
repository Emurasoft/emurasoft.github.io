# ShowLineNumbers 属性 (PrintProp 对象)

与配置属性中 [打印 页面](../../dlg/properties/print/index) 上的打印行号 复选框相对应。

## 

### \[JavaScript\]

```
b = object.ShowLineNumbers;
object.ShowLineNumbers = b;
```

### \[VBScript\]

```
b = object.ShowLineNumbers
object.ShowLineNumbers = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
