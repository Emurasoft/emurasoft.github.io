# UndoBufferSize 属性 (GeneralProp 对象)

与配置属性中 [**常规** 页面](../../dlg/properties/general/index) 上的 **最大撤消次数** 文本框相对应。

## 

### \[JavaScript\]

```
b = object.UndoBufferSize;
object.UndoBufferSize = b;
```

### \[VBScript\]

```
b = object.UndoBufferSize
object.UndoBufferSize = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
