# LineNumbersZeroBase 属性 (GeneralProp 对象)

与配置属性中 [**常规** 页面](../../dlg/properties/general/index) 上的 **显示从 0 开始的行/列号** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.LineNumbersZeroBase;
object.LineNumbersZeroBase = b;
```

### \[VBScript\]

```
b = object.LineNumbersZeroBase
object.LineNumbersZeroBase = b
```

## 版本

支持 EmEditor Professional 26.1 或更高的版本。
