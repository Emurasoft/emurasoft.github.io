# ShowControlCharacters 属性

与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示控制字符** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **ShowControlCharacters**;

object. **ShowControlCharacters** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowControlCharacters**

object. **ShowControlCharacters** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
