# CrLfDifferent 属性

与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **使用不同的标记来显示 CR 与 LF** 复选框相对应。

#### \[JavaScript\]

_b_ = object. **CrLfDifferent**;

object. **CrLfDifferent** = _b_;

#### \[VBScript\]

_b_ = object. **CrLfDifferent**

object. **CrLfDifferent** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
