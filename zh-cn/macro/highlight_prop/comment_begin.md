# CommentBegin 属性

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **“注释”群组框 \- “开始”** 文本框相对应。

#### \[JavaScript\]

_str_ =
object. **CommentBegin**;

object. **CommentBegin** = _str_;

#### \[VBScript\]

_str_ =
object. **CommentBegin**

object. **CommentBegin** = _str_

## 版本

支持 EmEditor 7.00 或之后的版本。
