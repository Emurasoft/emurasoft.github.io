# CommentEnd 属性

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **“注释”群组框 \- “结束”** 文本框相对应。

#### \[JavaScript\]

_b_ =
object. **CommentEnd**;

object. **CommentEnd** = _b_;

#### \[VBScript\]

_b_ =
object. **CommentEnd**

object. **CommentEnd** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
