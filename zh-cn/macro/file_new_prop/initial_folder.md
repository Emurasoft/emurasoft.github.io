# InitialFolder 属性 (FileNewProp 对象)

与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **初始文件夹** 文本框相对应。

## 

### \[JavaScript\]

```
s = object.InitialFolder;
object.InitialFolder = s;
```

### \[VBScript\]

```
s = object.InitialFolder
object.InitialFolder = s
```

## 版本

支持 EmEditor 14.1 或之后的版本。
