# UseTemplate 属性

与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **使用模板** 文本框相对应。

#### \[JavaScript\]

_b_ =
object. **UseTemplate**;

object. **UseTemplate** = _b_;

#### \[VBScript\]

_b_ =
object. **UseTemplate**

object. **UseTemplate** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
