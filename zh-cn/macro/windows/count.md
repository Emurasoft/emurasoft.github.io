# Count 属性 (Windows 集合)

检索窗口数目。

## 

### \[JavaScript\]

```
nCount = shell.windows.Count;
```

### \[VBScript\]

```
nCount = shell.windows.Count
```

## 示例

### \[JavaScript\]

```
alert( "Number of top-level windows: " + shell.windows.Count );
```

### \[VBScript\]

```
alert "Number of top-level windows: " & shell.windows.Count
```

## 版本

支持 EmEditor 7.00 或之后的版本。
