# 重复行命令

## 摘要

重复当前逻辑行。

## 说明

在当前光标位置下方插入当前逻辑行的副本。

## 运行方法

- 默认菜单: **插入** \> **重复行**
- [所有命令](../tools/all_commands): **插入** \> **重复行**
- 工具栏: ![](../../images/duplicateline.png)
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+Y

## 插件命令ID

```
EEID_DUPLICATE_LINE (4328)
```

## 宏

### \[JavaScript\]

```
document.selection.DuplicateLine();
```

### \[VBScript\]

```
document.selection.DuplicateLine
```
