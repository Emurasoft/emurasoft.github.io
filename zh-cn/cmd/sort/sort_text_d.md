# 按字母降序排序命令

## 摘要

在当前列中以字母降序排序文本。

## 说明

在当前列中以字母降序排序文本。如果文本不包含字母字符（A-Z和a-z），这个命令会尝试按数字值排序。

## 运行方法

- 默认菜单: **排序** \> **按字母降序排序**
- [所有命令](../tools/all_commands): **排序** \> **按字母降序排序**
- 工具栏: ![](../../images/sortingz-a.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_TEXT_D (4478)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4478);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4478
```
