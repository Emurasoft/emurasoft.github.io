# 高级排序命令

## 摘要

排序文档中的多个列或用高级选项。

## 说明

排序文档中的多个列或用高级选项。

## 运行方法

- 默认菜单: **排序** \> **高级排序**
- [所有命令](../tools/all_commands): **排序** \> **高级排序**
- 工具栏: ![](../../images/sortbymultiple..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SORT\_MULTI (3903)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3903);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3903
```
