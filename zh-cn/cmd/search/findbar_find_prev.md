# 查找上一个 (查找工具栏) 命令

## 摘要

用查找工具栏查找上一个匹配。

## 说明

用查找工具栏查找上一个匹配。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **查找工具栏** \> **查找上一个**
- 工具栏: ![](../../images/editrepeatback..png) (查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_FIND_PREV (4569)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4569);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4569
```
