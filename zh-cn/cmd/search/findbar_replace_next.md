# 替换下一个 (查找工具栏) 命令

## 摘要

用查找工具栏替换下一个匹配的字符串。

## 说明

用查找工具栏替换下一个匹配的字符串。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **查找工具栏** \> **替换下一个**
- 工具栏: ![](../../images/replace.png) (查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_REPLACE_NEXT (4581)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4581);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4581
```
