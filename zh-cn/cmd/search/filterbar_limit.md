# 阻止多行更改 (筛选工具栏) 命令

## 摘要

切换“筛选”工具栏上的「阻止多行更改」按钮。

## 说明

切换“筛选”工具栏上的「阻止多行更改」按钮。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **筛选工具栏** \> **阻止多行更改**
- 工具栏: ![](../../images/blockmultilinechanges..png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FILTERBAR\_LIMIT(3915)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3915);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3915
```
