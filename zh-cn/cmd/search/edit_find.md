# 查找命令

## 摘要

查找一个字符串。

## 说明

查找一个指定的字符串。这个命令显示 [查找对话框](../../dlg/find/index)。指定一个字符串和其他参数会，就可以开始在文件中查找指定的字符串。

## 运行方法

- 默认菜单: **搜索** \> **查找**
- [所有命令](../tools/all_commands): **搜索**
\> **查找**
- 工具栏: ![](../../images/editfind.png)
- 状态栏: 无
- 默认快捷键: CTRL+F

## 插件命令ID

```
EEID_EDIT_FIND (4200)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4200);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4200
```
