# 高级 (查找工具栏) 命令

## 摘要

显示查找工具栏的高级对话框。

## 说明

显示查找工具栏的高级对话框。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **查找工具栏** \> **高级**
- 工具栏: ![](../../images/commonsettings..png)
(查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FINDBAR\_ADVANCED (3923)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3923);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3923
```
