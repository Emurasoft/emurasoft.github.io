# 刷新命令

## 摘要

用当前筛选设定刷新文档。

## 说明

用当前筛选设定刷新文档。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>筛选工具栏 \>刷新
- 工具栏: ![](../../images/refresh.gif) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FILTERBAR\_REFRESH (3905)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3905);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3905
```
