# 模糊匹配 (筛选工具栏) 命令

## 摘要

切换“筛选”工具栏上的「模糊匹配」按钮。

## 说明

切换“筛选”工具栏上的「模糊匹配」按钮。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **筛选工具栏** \> **模糊匹配**
- 工具栏:  ![](../../images/fuzzy..png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILTERBAR_FUZZY (23235)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23235);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23235
```
