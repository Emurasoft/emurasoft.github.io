# 同步滚动命令

## 摘要

不指定参数直接同步滚动两个最近访问的文档。

## 说明

不指定参数直接同步滚动两个最近访问的文档。

## 运行方法

- 默认菜单: **比较** \> **同步滚动**
- [所有命令](../tools/all_commands): **比较** \> **同步滚动**
- 工具栏:  ![](../../images/sync24x16..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SYNC_SCROLL_DIRECT (4495)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4495);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4495
```
