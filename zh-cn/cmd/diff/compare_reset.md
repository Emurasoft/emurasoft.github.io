# 重置命令

## 摘要

重置比较或同步滚动模式并清除比较结果。

## 说明

重置比较或同步滚动模式并清除比较结果。（文件还原成标签模式）。

## 运行方法

- 默认菜单: **比较** \> **重置为普通模式**
- [所有命令](../tools/all_commands): **比较** \> **重置为普通模式**
- 工具栏: ![](../../images/reset24x16..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COMPARE_RESET (4497)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4497);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4497
```
