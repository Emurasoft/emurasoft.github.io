# 重新扫描命令

## 摘要

重新扫描并刷新所比较的文档。

## 说明

重新扫描被比较的文档并更新结果。

## 运行方法

- 默认菜单: **比较** \> **重新扫描**
- [所有命令](../tools/all_commands): **比较** \> **重新扫描**
- 工具栏:  ![](../../images/rescan24x16.png)
- 状态栏: 无
- 默认快捷键: **F5**

## 插件命令ID

```
EEID_COMPARE_RESCAN (4494)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4494);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4494
```
