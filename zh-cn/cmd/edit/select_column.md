# 选择不包括标题的 CSV 列命令

## 摘要

在 CSV 文档中用垂直选择模式选择当前列但不包括标题。

## 说明

在 CSV 文档中用垂直选择模式选择当前列但不包括标题。

## 运行方法

- 默认菜单: **CSV** \> **列** \> **选择不包括标题的列**
- [所有命令](../tools/all_commands): **CSV** \> **列** \> **选择不包括标题的列**
- 工具栏: ![](../../images/columns_separators.gif) \+ 选择不包括标题的列
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SELECT_COLUMN (4461)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4461);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4461
```
