# 斜体命令

## 摘要

切换斜体样式。

## 说明

切换斜体样式。

## 运行方法

- 默认菜单: **编辑** \> **Markdown/HTML** \> **斜体**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown/HTML** \> **斜体**
- 工具栏: ![](../../images/italic.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_ITALIC (23270)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23270);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23270
```
