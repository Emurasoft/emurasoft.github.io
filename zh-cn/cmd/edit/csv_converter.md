# CSV 转换器命令

## 摘要

显示或隐藏 CSV 转换器。

## 说明

显示或隐藏 CSV 转换器。

## 运行方法

- 默认菜单:CSV \>CSV 转换器
- [所有命令](../tools/all_commands):CSV \>CSV 转换器
- 工具栏: ![](../../images/convert_to.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_CSV\_CONVERTER (3891)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3891);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3891
```
