# 代码命令

## 摘要

切换代码样式。

## 说明

切换代码样式。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **代码**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **代码**
- 工具栏: ![](../../images/markdown_code.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_CODE (23278)
```

## 宏

```
editor.ExecuteCommandByID(23278);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23278
```
