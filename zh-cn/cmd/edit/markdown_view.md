# Markdown 设计视图命令

## 摘要

显示 Markdown 设计视图。

## 说明

显示 Markdown 设计视图。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **设计视图**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **设计视图**
- 工具栏: ![](../../images/markdown_view.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_VIEW (23255)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23255);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23255
```
