# 粗体命令

## 摘要

切换粗体样式。

## 说明

切换粗体样式。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **粗体**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **粗体**
- 工具栏: ![](../../images/bold..png)
- 状态栏: 无
- 默认快捷键: CTRL+B

## 插件命令 ID

```
EEID_MARKDOWN_BOLD (23269)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23269);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23269
```
