# 全部撤消命令

## 摘要

撤消所有之前的操作。

## 说明

撤消所有之前的操作。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>撤消 \>全部撤消
- 工具栏: ![](../../images/editundo.gif) (向下箭头)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_UNDO_ALL (3980)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3980);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3980
```
