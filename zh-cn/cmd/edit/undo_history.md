# 撤消/重做历史命令

## 摘要

显示 [**撤消/重做历史** 对话框](../../dlg/undo_history/index)。

## 说明

显示 [**撤消/重做历史** 对话框](../../dlg/undo_history/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **撤消** \> **撤消/重做历史**
- 工具栏: ![](../../images/editundo.png) (向下箭头)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_UNDO_HISTORY (3982)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3982);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3982
```
