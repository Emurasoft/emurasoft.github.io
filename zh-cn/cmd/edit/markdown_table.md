# 表格命令

## 摘要

插入表格。

## 说明

插入表格。

## 运行方法

- 默认菜单: **编辑** \> **Markdown/HTML** \> **表格**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown/HTML** \> **表格**
- 工具栏: ![](../../images/table.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_TABLE (23273)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23273);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23273
```
