# 四级标题命令

## 摘要

将当前行设置为四级标题。

## 说明

将当前行设置为四级标题。

## 运行方法

- 默认菜单: **编辑** \> **Markdown/HTML** \> **四级标题**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown/HTML** \> **四级标题**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_HEADING4 (23260)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23260);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23260
```
