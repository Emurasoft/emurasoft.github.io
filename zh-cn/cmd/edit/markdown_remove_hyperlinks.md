# 移除超链接命令

## 摘要

移除超链接。

## 说明

移除超链接。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **移除超链接**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **移除超链接**
- 工具栏: ![](../../images/remove_hyperlinks..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_REMOVE_HYPERLINKS (23272)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23272);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23272
```
