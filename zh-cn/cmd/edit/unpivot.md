# 逆透视命令

## 摘要

通过平展 CSV 数据将列转换为行。

## 说明

该命令会显示 [**逆透视** 对话框](../../dlg/unpivot/index)，并通过平展 CSV 数据将列转换为行。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **逆透视**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **逆透视**
- 工具栏: ![](../../images/unpivot.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_UNPIVOT (4083)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4083);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4083
```
