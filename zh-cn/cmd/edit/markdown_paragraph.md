# 段落命令

## 摘要

将当前行设置为单个段落。

## 说明

将当前行设置为单个段落。

## 运行方法

- 默认菜单: **编辑** \> **Markdown/HTML** \> **段落**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown/HTML** \> **段落**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_PARAGRAPH (23256)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23256);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23256
```
