# 换行符命令

## 摘要

插入换行符。

## 说明

插入换行符。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **换行符**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **换行符**
- 工具栏: ![](../../images/markdown_line_break..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_LINE_BREAK (23268)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23268);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23268
```
