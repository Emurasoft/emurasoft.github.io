# 图像命令

## 摘要

插入一个图像。

## 说明

插入一个图像。

## 运行方法

- 默认菜单: **编辑** \> **Markdown/HTML** \> **图像**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown/HTML** \> **图像**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_IMAGE (23277)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23277);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23277
```
