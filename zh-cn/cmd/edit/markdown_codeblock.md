# 代码块命令

## 摘要

将当前行设置为代码块。

## 说明

将当前行设置为代码块。

## 运行方法

- 默认菜单: **编辑** \> **Markdown/HTML** \> **代码块**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown/HTML** \> **代码块**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_CODEBLOCK (23264)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23264);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23264
```
