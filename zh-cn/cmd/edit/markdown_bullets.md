# 无序列表命令

## 摘要

插入无序列表。

## 说明

插入无序列表。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **无序列表**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **无序列表**
- 工具栏: ![](../../images/bullets..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_BULLETS (23266)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23266);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23266
```
