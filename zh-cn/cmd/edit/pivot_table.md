# 数据透视表命令

## 摘要

在 CSV 文档中创建数据透视表。

## 说明

该命令会显示 [**数据透视表** 对话框](../../dlg/pivot_table/index)，并在 CSV 文档中创建数据透视表。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **数据透视表**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **数据透视表**
- 工具栏: ![](../../images/pivot_table.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_PIVOT\_TABLE (4081)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4081);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4081
```
