# 超链接命令

## 摘要

插入超链接。

## 说明

插入超链接。

## 运行方法

- 默认菜单: **编辑** \> **Markdown** \> **超链接**
- [所有命令](../tools/all_commands): **编辑** \> **Markdown** \> **超链接**
- 工具栏: ![](../../images/hyperlink.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_HYPERLINK (23271)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23271);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23271
```
