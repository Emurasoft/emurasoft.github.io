# 全部重做命令

## 摘要

重做所有之前的操作。

## 说明

重做所有之前的操作。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **重做** \> **全部重做**
- 工具栏: ![](../../images/editredo.png) (向下箭头)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_REDO_ALL (3981)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3981);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3981
```
