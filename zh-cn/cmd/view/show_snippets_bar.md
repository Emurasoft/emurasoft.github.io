# 代码片段工具栏命令

## 摘要

显示或隐藏代码片段工具栏。

## 说明

显示或隐藏代码片段工具栏。

## 运行方法

- 默认菜单：**查看** > **工具栏** \> **代码片段**
- [所有命令](../tools/all_commands)：**查看** > **工具栏** \> **代码片段**
- 工具栏：无
- 状态栏：无
- 默认快捷键：无

## 插件命令ID

```
EEID_SHOW_SNIPPETS_BAR (23287)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23287);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23287
```
