# 布局 1（焦点模式）命令

## 摘要

切换布局 1（默认为焦点模式）模式。

## 说明

切换布局 1（默认为焦点模式）模式。

## 运行方法

- 默认菜单: **查看** \> **布局** \> **全屏**
- [所有命令](../tools/all_commands): **查看** \> **布局** \> **全屏**
- 工具栏: ![](../../images/full_screen.png)
- 状态栏: 无
- 默认快捷键: F11

## 插件命令ID

```
EEID_FULL_SCREEN (4453)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4453);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4453
```
