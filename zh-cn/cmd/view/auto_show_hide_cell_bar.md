# 自动显示/隐藏单元格工具栏命令

## 摘要

自动显示或隐藏单元格工具栏。

## 说明

自动显示或隐藏单元格工具栏。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** >
**工具栏** \> **单元格工具栏选项** \> **自动显示/隐藏单元格工具栏**
- 工具栏: ![](../../images/commonsettings..png) (单元格工具栏) \+ 自动显示/隐藏单元格工具栏
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CELL_BAR_AUTO_SHOW (3969)
```

## 宏

editor.ExecuteCommandByID(3969);

### \[VBScript\]

```
editor.ExecuteCommandByID 3969
```
