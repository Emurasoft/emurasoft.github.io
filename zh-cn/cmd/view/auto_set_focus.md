# 自动设置焦点（单元格工具栏）命令

## 摘要

自动设置焦点到单元格工具栏上。

## 说明

自动设置焦点到单元格工具栏上。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** >
**工具栏** \> **单元格工具栏选项** \> **自动设置焦点**
- 工具栏: ![](../../images/commonsettings..png) (单元格工具栏) \+ 自动设置焦点
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_CELL_BAR_AUTO_FOCUS (3970)
```

## 宏

editor.ExecuteCommandByID(3970);

### \[VBScript\]

```
editor.ExecuteCommandByID 3970
```
