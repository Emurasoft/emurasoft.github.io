# 自动调整高度（单元格工具栏）命令

## 摘要

自动调整单元格工具栏的高度。

## 说明

自动调整单元格工具栏的高度。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** >
**工具栏** \> **单元格工具栏选项** \> **自动调整高度**
- 工具栏: ![](../../images/commonsettings.gif) (单元格工具栏) \+ 自动调整高度
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CELL_BAR_AUTO_HEIGHT (3971)```

## 宏

editor.ExecuteCommandByID(3971);

### \[VBScript\]

```
editor.ExecuteCommandByID 3971
```
