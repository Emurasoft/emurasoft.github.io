# 标尺命令

## 摘要

显示或隐藏标尺。

## 说明

显示或隐藏标尺。在默认设定下，EmEditor 不显示标尺。

## 运行方法

- 默认菜单: **查看** \> **标尺**
- [所有命令](../tools/all_commands): **查看** \> **标尺**
- 工具栏:
![](../../images/ruler24x16.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_RULER (4531)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4531);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4531
```
