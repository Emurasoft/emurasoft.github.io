# 自定义字体命令

## 摘要

更改字体设定。

## 说明

这个命令显示 [**自定义字体** 对话框](../../dlg/properties/font/index)，让你可以设定显示的字体和打印字体。

## 运行方法

- 默认菜单: **查看** \> **字体**
- [所有命令](../tools/all_commands): **查看** >
**字体** \> **字体**
- 工具栏: ![](../../images/fontpopup.gif)（点击箭头）
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FONT (4218)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4218);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4218
```
