# 使用 DirectWrite 命令

## 摘要

指定是否使用 DirectWrite 进行文本呈现。

## 说明

指定是否使用 DirectWrite 进行文本呈现。

## 运行方法

- 默认菜单: **查看** \> **使用 DirectWrite**
- [所有命令](../tools/all_commands): **查看** \> **字体** \> **使用 DirectWrite**
- 工具栏: ![](../../images/fontpopup.gif)
（点击箭头） \> **使用 DirectWrite**
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_USE_DIRECTWRITE (3997)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3997);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3997
```
