# 切换筛选工具栏命令

## 摘要

显示或隐藏筛选工具栏。

## 说明

显示或隐藏筛选工具栏。

## 运行方法

- 默认菜单: **查看** > **工具栏** \> **筛选**
- [所有命令](../tools/all_commands): **查看** > **工具栏** \> **筛选**
- 工具栏: ![](../../images/togglefilterbar.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SHOW_FILTER_BAR (3911)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3911);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3911
```
