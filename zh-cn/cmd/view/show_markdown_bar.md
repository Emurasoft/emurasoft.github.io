# Markdown 工具栏命令

## 摘要

显示或隐藏 Markdown 工具栏。

## 说明

显示或隐藏 Markdown 工具栏。

## 运行方法

- 默认菜单: **查看** > **工具栏** \> **Markdown**
- [所有命令](../tools/all_commands): **查看** > **工具栏** \> **Markdown**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_SHOW_MARKDOWN_BAR (23274)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23274);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23274
```
