# Markdown 转纯文本命令

## 摘要

将选取的文本从 Markdown 转换为纯文本。

## 说明

将选取的文本从 Markdown 转换为纯文本。

## 运行方法

- 默认菜单: **转换** \> **编码/解码所选内容** \> **Markdown 转纯文本**
- [所有命令](../tools/all_commands): **转换** \> **编码/解码所选内容** \> **Markdown 转纯文本**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_MARKDOWN_TO_TEXT (23282)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(23282);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23282
```
