# HTML 转 Markdown 命令

## 摘要

将选取的文本从 HTML 转换为 Markdown。

## 说明

将选取的文本从 HTML 转换为 Markdown。

## 运行方法

- 默认菜单: **转换** \> **编码/解码所选内容** \> **HTML 转 Markdown**
- [所有命令](../tools/all_commands): **转换** \> **编码/解码所选内容** \> **HTML 转 Markdown**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_HTML_TO_MARKDOWN (23281)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(23281);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23281
```
