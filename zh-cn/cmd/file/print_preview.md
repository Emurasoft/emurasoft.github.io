# 打印预览命令

## 摘要

显示当前文档的打印预览。

## 说明

显示 **打印预览** 窗口预览当前文档的打印效果。在 **打印预览** 窗口中，你可以打印或设置打印参数。

## 运行方法

- 默认菜单: **文件** \> **打印预览**
- [所有命令](../tools/all_commands): **文件** \> **打印**
\> **打印预览**
- 工具栏: ![](../../images/printpreview..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PRINT_PREVIEW (4286)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4286);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4286
```
