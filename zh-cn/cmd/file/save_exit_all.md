# 保存并全部关闭命令

## 摘要

保存并关闭所有打开的文件。

## 说明

保存并关闭所有打开的文件。这个命令等同于 [**全部保存** 命令](file_save_all) 加 [**全部关闭** 命令](exit_all)。

## 运行方法

- 默认菜单: **文件** \> **保存并全部关闭**
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **保存并全部关闭**
- 工具栏: ![](../../images/saveexitall.png)
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+E

## 插件命令ID

```
EEID_SAVE_EXIT_ALL (4118)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4118);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4118
```
