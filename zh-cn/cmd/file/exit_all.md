# 全部关闭命令

## 摘要

关闭打所有开的文件。

## 说明

这个命令会关闭所有已打开的文件。如果存在一个已被修改的文档，则会显示一个即时消息来让你选择是否想要保存变更。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **全部关闭**
- 工具栏: ![](../../images/exitall.png)
- 状态栏: 无
- 默认快捷键: ALT+SHIFT+X

## 插件命令ID

```
EEID_EXIT_ALL (4119)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4119);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4119
```
