# 保存并关闭命令

## 摘要

保存并关闭当前文件。

## 说明

把变更保存为，并关闭当前文档。

## 运行方法

- 默认菜单: **文件** \> **保存并关闭**
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **保存并关闭**
- 工具栏: ![](../../images/filesaveexit..png)
- 状态栏: 无
- 默认快捷键: CTRL+E

## 插件命令ID

```
EEID_FILE_SAVE_EXIT (4116)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4116);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4116
```
