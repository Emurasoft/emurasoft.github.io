# 清除当前文档中的所有书签命令

## 摘要

清除当前文档中的所有书签。

## 说明

清除当前文档中的所有书签。

## 运行方法

- 默认菜单: **书签** \> **清除所有书签**
- [所有命令](../tools/all_commands): **书签** \> **清除所有书签**
- 工具栏: ![](../../images/bookmarkclear..png)
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+F2

## 插件命令ID

```
EEID_BOOKMARK_CLEAR (4323)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4323);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4323
```
