# 新提示命令

## 摘要

输入一个新提示来询问 AI。

## 说明

显示[**新提示**对话框](../../dlg/new_prompt/index)以创建一个新的提示。

## 运行方法

- 默认菜单: **AI** \> **新提示**
- [所有命令](../tools/all_commands): **AI** \> **新提示**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_NEW_PROMPT (23252)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23252);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23252
```
