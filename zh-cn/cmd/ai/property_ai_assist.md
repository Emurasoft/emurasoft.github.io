# AI 辅助属性命令

## 摘要

当前配置的 AI 辅助属性。

## 说明

显示 **[配置属性](../../dlg/properties/index)** 对话框中的 [**AI 辅助** 页面](../../dlg/properties/ai_assist/index)。

## 运行方法

- 默认菜单: **AI** \> **AI 辅助属性**
- [所有命令](../tools/all_commands): **AI** \> **AI 辅助属性**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_AI_ASSIST (8983)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(8983);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8983
```
