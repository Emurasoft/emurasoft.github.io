# 自定义与 AI 聊天命令

## 摘要

打开自定义对话框中的与 AI 聊天页面。

## 说明

显示让你能定义通用应用设置的[**自定义**对话框](../../dlg/customize/index)中的 **与 AI 聊天** 页面。

## 运行方法

- 默认菜单：**AI** > **自定义** > **与 AI 聊天**
- [所有命令](all_commands)：**AI** > **自定义** > **与 AI 聊天**
- 工具栏：无
- 状态栏：无
- 默认快捷键：无

## 插件命令ID

```
EEID_CUSTOMIZE_CHAT_AI (9074)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(9074);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9074
```
