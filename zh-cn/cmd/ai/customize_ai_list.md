# 自定义 AI 提示命令

## 摘要

打开自定义对话框中的 AI 提示页面。

## 说明

显示让你能定义通用应用设置的[**自定义**对话框](../../dlg/customize/index)中的 **AI 提示** 页面。

## 运行方法

- 默认菜单：**AI** > **自定义** > **AI 提示**
- [所有命令](all_commands)：**AI** > **自定义** > **AI 提示**
- 工具栏：无
- 状态栏：无
- 默认快捷键：无

## 插件命令 ID

```
EEID_CUSTOMIZE_AI_LIST (9073)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(9073);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9073
```
