# 定义配置命令

## 摘要

定义配置。

## 说明

显示 [**定义配置** 对话框](../../dlg/configurations/index)，让你能定义新配置或显示当前以及选取的配置属性。

## 运行方法

- 默认菜单: **工具** \> **选择配置** \> **定义配置**
- [所有命令](all_commands): **工具** >
**Select Configuration** \> **定义配置**
- 工具栏: ![](../../images/configpopup.png)（点击箭头）
- 状态栏: （双击配置名称） \> **定义配置**
- 默认快捷键: 无

## 插件命令ID

```
EEID_CONFIG (4221)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4221);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4221
```
