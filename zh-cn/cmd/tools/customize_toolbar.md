# 自定义工具栏命令

## 摘要

自定义工具栏设置。

## 说明

显示**自定义**对话框的[**工具栏按钮**](../../dlg/customize/toolbar_buttons/index)页面，您可以在此自定义工具栏。

## 运行方法

- 默认菜单: **工具** \> **自定义工具栏**
- [所有命令](all_commands): **工具** >
**自定义工具栏**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CUSTOMIZE_TOOLBAR (4224)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4224);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4224
```
