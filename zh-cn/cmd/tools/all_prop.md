# 所有配置的属性命令

## 摘要

更改所有配置的属性。

## 说明

为所有配置显示 **[配置属性](../../dlg/properties/index)** 对话框。

## 运行方法

- 默认菜单: **工具** \> **所有配置的属性**
- [所有命令](all_commands): **工具** >
**所有配置的属性**
- 工具栏: ![](../../images/allproperties.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_ALL_PROP (4270)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4270);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4270
```
