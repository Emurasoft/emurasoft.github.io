# 自定义代码片段命令

## 摘要

打开自定义对话框的代码片段页面。

## 说明

显示[**自定义** 对话框](../../dlg/customize/index)的代码片段页面，您可以在其中定义应用程序的通用设置。

## 运行方法

- 默认菜单：无
- [所有命令](all_commands)：**工具** > **自定义** > **代码片段**
- 工具栏：无
- 状态栏：无
- 默认快捷键：无

## 插件命令 ID

```
EEID_CUSTOMIZE_SNIPPETS (9081)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(9081);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9081
```
