# 当前配置属性命令

## 摘要

更改当前选定配置的属性。

## 说明

为当前配置显示 **[配置属性](../../dlg/properties/index)** 对话框。在这个菜单中，你可以为当前配置设置不同的参数。

## 运行方法

- 默认菜单: **工具** \> **当前配置属性**
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **属性**
- 工具栏: ![](../../images/properties..png)
- 状态栏: 无
- 默认快捷键: ALT+ENTER

## 插件命令ID

```
EEID_CUSTOMIZE (4219)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4219);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4219
```
