# 在左边插入一列命令

## 摘要

在含有分隔值的文档中的当前列的左边插入新的一列。

## 说明

在含有分隔值的文档中的当前列的左边插入新的一列。

## 运行方法

- 默认菜单: **CSV** \> **列** \> **在左边插入一列**
- [所有命令](../tools/all_commands): **CSV** \> **列** \> **在左边插入一列**
- 工具栏: ![](../../images/columns_separators.png) \+ 在左边插入一列
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_INSERT\_COLUMN (3887)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3887);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3887
```
