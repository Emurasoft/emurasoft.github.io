# 删除所有分隔符命令

## 摘要

在标准模式下，删除所有分隔符。

## 说明

在标准模式下，删除所有分隔符。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **CSV** \> **分隔符** \> **删除所有分隔符**
- 工具栏: ![](../../images/columns_separators..png) \+ 删除所有分隔符
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_REMOVE\_ALL\_SEPARATOR (3886)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3886);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3886
```
