# 列/分隔符 (弹出菜单) 命令

## 摘要

显示一个设置列或分隔符选项的弹出式菜单。

## 说明

显示一个设置列或分隔符选项的弹出式菜单。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **CSV** \> **分隔符** \> **列/分隔符 (弹出菜单)**
- 工具栏: ![](../../images/columns_separators.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SV\_SEP\_POPUP (3893)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3893);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3893
```
