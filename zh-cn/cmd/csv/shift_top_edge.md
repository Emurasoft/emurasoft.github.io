# 扩展到上边缘 (仅限单元格选择模式) 命令

## 摘要

把选区扩展到当前数据区域的上边缘。

## 说明

把选区扩展到当前数据区域的上边缘。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **CSV** \> **仅限单元格选择模式** \> **扩展到上边缘**
- 工具栏: ![](../../images/cell_selection_mode..png)
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+向上键 (仅限单元格选择模式)

## 插件命令ID

```
EEID_SHIFT_TOP_EDGE (4023)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4023);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4023
```
