# 单元格选择模式命令

## 摘要

在 CSV 文档中编辑当前单元格。

## 说明

在 CSV 文档中编辑当前单元格。

## 运行方法

- 默认菜单: **编辑** \> **CSV** \> **单元格选择模式**
- [所有命令](../tools/all_commands): **编辑** \> **CSV** \> **单元格选择模式**
- 工具栏: ![](../../images/cell_selection_mode.gif)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CSV_CELL_MODE (3959)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3959);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3959
```
