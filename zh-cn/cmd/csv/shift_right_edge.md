# 扩展到右边缘 (仅限单元格选择模式) 命令

## 摘要

把选区扩展到当前数据区域的右边缘。

## 说明

把选区扩展到当前数据区域的右边缘。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **CSV** \> **仅限单元格选择模式** \> **扩展到右边缘**
- 工具栏: ![](../../images/cell_selection_mode.png)
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+向右键 (仅限单元格选择模式)

## 插件命令ID

```
EEID_SHIFT_RIGHT_EDGE (4021)
```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4021);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4021
```
