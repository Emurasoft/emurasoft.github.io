# 调整分隔符位置 (仅可见行) 命令

## 摘要

仅在 CSV 文档的可见行上调整分隔符位置。

## 说明

仅在 CSV 文档的可见行上调整分隔符位置。

## 运行方法

- 默认菜单: **CSV** \> **调整分隔符位置 (仅可见行)**
- [所有命令](../tools/all_commands): **CSV** \> **调整分隔符位置 (仅可见行)**
- 工具栏: ![](../../images/columns_separators.png) \+ 调整分隔符位置 (仅可见行)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_ADJUST\_COLUMN\_VISIBLE (3901)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3901);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3901
```
