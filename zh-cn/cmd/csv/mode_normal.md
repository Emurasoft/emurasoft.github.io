# 标准模式命令

## 摘要

选择标准模式。

## 说明

清除CSV (逗号分隔)，TSV (制表符分隔)，和DSV (用户定义的分隔符分隔) 模式。

## 运行方法

- 默认菜单: **编辑** \> **CSV** \> **标准模式**
- [所有命令](../tools/all_commands): **编辑** \> **CSV** \> **标准模式**
- 工具栏: ![](../../images/default_mode..png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MODE_NORMAL (4472)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4472);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4472
```
