# 转置命令

## 摘要

切换 CSV 文档的行和列。

## 说明

切换 CSV 文档的行和列。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **转置**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **转置**
- 工具栏: ![](../../images/transpose.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_TRANSPOSE (4082)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4082);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4082
```
