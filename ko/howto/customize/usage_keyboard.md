# 키보드 바로가기 바꾸기

1. **키보드 탭** 을 디스플레이 하기 위해서 ![Properties for Current Configuration](../../images/properties.gif)
(현재 구성만을 바꾸려면) 혹은
![Properties for All Configuration](../../images/allproperties.gif)
(모든 구성을 바꾸려면) 을 선택해 주십시오.
2. 범위와 명령을 선택하십시오.
3. **새로운 바로가기 키를 입력** 에서 바로가기 키를 입력해 주십시오.
4. **추가** 버튼을 클릭해 주십시오.
