# 저장 후 닫기 명령

## 요약

현재 파일을 저장 후 닫습니다.

## 설명

변경 사항을 저장하고 현재 문서를 닫습니다.

## 실행하는 방법

- 기본 메뉴: **파일** \> **저장 후 닫기**
- [모든 명령](../tools/all_commands): **파일** \> **닫기**
\> **저장 후 닫기**
- 도구 모음: ![](../../images/filesaveexit.png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+E

## 플러그인 명령 ID

```
EEID_FILE_SAVE_EXIT (4116)
```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4116);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4116
```
