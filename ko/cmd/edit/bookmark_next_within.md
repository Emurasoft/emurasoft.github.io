# 이 문서의 다음 책갈피 명령

### 요약

> 이 문서내의 다음 책갈피로 이동합니다.

### 설명

> 커서를 현재 문서내의 다음 책갈피로 이동합니다.
> 현재 문서에 책갈피가 없다면 이 명령은 커서를 움직이지 않습니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **책갈피** \> **다음 책갈피**
- [모든 명령](../tools/all_commands): **편집** \> **책갈피** \> **다음 책갈피**
- 도구 모음: ![](../../images/bookmarknextwithin.gif)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_BOOKMARK\_NEXT\_WITHIN (4351)

### 매크로

#### \[JavaScript\]

> document.selection.NextBookmark();

#### \[VBScript\]

> document.selection.NextBookmark
