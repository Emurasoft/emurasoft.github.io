# 사용자 정의 DSV 모드 명령

## 요약

사용자 정의 DSV (구분기호로 구분) 모드를 선택합니다.

## 설명

사용자 정의 DSV (구분기호로 구분) 모드를 선택합니다.

## 실행하는 방법

- 기본 메뉴: **편집** \> **구분된 값/정렬** \> **사용자 정의 DSV 모드**
- [모든 명령](../tools/all_commands): **편집** \> **구분된 값/정렬** \> **사용자 정의 DSV 모드**
- 도구 모음: ![](../../images/dsv.png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_MODE_DSV (4476)
```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4476);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4476
```
