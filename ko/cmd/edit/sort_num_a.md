# 오름차순 정렬 명령

## 요약

오름차순으로 현재 열의 숫자를 정렬합니다.

## 설명

오름차순으로 현재 열의 숫자를 정렬합니다.
이 명령은 숫자가 아닌 문자에 대해서는 작동하지 않습니다.

## 실행하는 방법

- 기본 메뉴:편집 \>구분된 값/정렬 \>오름차순 정렬
- [모든 명령](../tools/all_commands):편집 \>구분된 값/정렬 \>오름차순 정렬
- 도구 모음: ![](../../images/sorting0-9.gif)
- 상태 표시줄: 없음
