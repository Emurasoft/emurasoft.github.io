# 지정되지 않은 텍스트 맞춤법 확인 명령

## 요약

지정되지 않은 텍스트의 맞춤법을 확인합니다.

## 설명

지정되지 않은 텍스트의 맞춤법을 확인합니다. 맞춤법이 틀린 단어는 빨간색으로 밑줄이 표시됩니다.

## 실행하는 방법

- 기본 메뉴: **편집** \> **맞춤법** \> **맞춤법 확인을 할 장소** \>
**지정되지 않음**
- [모든 명령](../tools/all_commands): **편집** \> **맞춤법** \> **맞춤법 확인을 할 장소** \>
**지정되지 않은 텍스트**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_SPELL_REST (4521)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4521);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4521
```
