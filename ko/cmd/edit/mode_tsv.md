# TSV 모드 명령

## 요약

TSV (탭으로 구분) 모드를 선택합니다.

## 설명

TSV (탭으로 구분) 모드를 선택합니다.

## 실행하는 방법

- 기본 메뉴: **편집** \> **구분된 값/정렬** \> **TSV 모드**
- [모든 명령](../tools/all_commands): **편집** \> **구분된 값/정렬** \> **TSV 모드**
- 도구 모음: ![](../../images/tsv..png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_MODE_TSV (4471)
```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4471);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4471
```
