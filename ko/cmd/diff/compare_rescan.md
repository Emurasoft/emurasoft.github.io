# 다시 검사 명령

### 요약

> 비교 문서를 다시 검사하여 결과를 새롭게 나타냅니다.

### 설명

> 비교 문서를 다시 검사하여 결과를 새롭게 나타냅니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **다시 검사**
- [모든 명령](../tools/all_commands): **비교** \> **다시 검사**
- 도구 모음: ![](../../images/rescan24x16.gif)
- 상태 표시줄: 없음
- 기본 바로 가기 키: **F5**

### 플러그인 명령 ID

- EEID\_COMPARE\_RESCAN (4494)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4494);

#### \[VBScript\]

> editor.ExecuteCommandByID 4494
