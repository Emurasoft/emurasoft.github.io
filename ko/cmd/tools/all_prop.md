# 모든 구성 속성 명령

## 요약

모든 구성의 속성을 변경합니다.

## 설명

모든 구성에 관한 **[구성 속성](../../dlg/properties/index)** 대화 상자를 표시합니다.

## 실행하는 방법

- 기본 메뉴: **도구** \> **모든 구성 속성**
- [모든 명령](all_commands): **도구** >
**모든 구성 속성**
- 도구 모음: ![](../../images/allproperties..png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_ALL_PROP (4270)
```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4270);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4270
```
