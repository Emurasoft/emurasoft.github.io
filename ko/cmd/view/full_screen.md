# 전체 화면 명령

## 요약

전체 화면 모드를 설정/해제 합니다.

## 설명

전체 화면 모드를 설정/해제 합니다.

## 실행하는 방법

- 기본 메뉴: **보기** \> **전체 화면**
- [모든 명령](../tools/all_commands): **보기** \> **전체 화면**
- 도구 모음: ![](../../images/full_screen..png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: F11

## 플러그인 명령 ID

```
EEID_FULL_SCREEN (4453)
```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4453);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4453
```
