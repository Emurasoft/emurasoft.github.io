# 눈금자 명령

## 요약

눈금자를 표시하거나 숨깁니다.

## 설명

눈금자를 표시하거나 숨깁니다.
기본적으로 엠에디터는 눈금자를 표시하지 않습니다.

## 실행하는 방법

- 기본 메뉴: **보기** \> **눈금자**
- [모든 명령](../tools/all_commands): **보기** \> **눈금자**
- 도구 모음:
![](../../images/ruler24x16.png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_VIEW_LINE_RULER (4531)
```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4531);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4531
```
