# 매크로 저장 명령

## 요약

녹화된 매크로를 다른 이름으로 저장합니다.

## 설명

[매크로 녹화 시작/중지 명령](quick_macro_record) 을 통해 이전에 녹화한 임시 매크로를 저장합니다.
이 명령은다른 이름으로 저장 대화 상자을 나타내고 파일 이름과 매크로 유형(JavaScript 또는 VBScript)을 입력할 수
있도록 합니다.

## 실행하는 방법

- 기본 메뉴:매크로 \>다른 이름으로 저장
- [모든 명령](../tools/all_commands):매크로
\>다른 이름으로 저장
- 도구 모음: ![](../../images/macrosave.gif)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_MACRO_SAVE (4346)```
