# 매크로 선택 명령

## 요약

매크로를 선택합니다.

## 설명

실행할 매크로를 선택합니다.
이 명령은 [**매크로 편집** 명령](macro_edit), [**매크로 실행** 명령](quick_macro_run) 과 결합하여 사용됩니다.

## 실행하는 방법

- 기본 메뉴: **매크로** \> **선택**
- [모든 명령](../tools/all_commands): **매크로**
\> **선택**
- 도구 모음: ![](../../images/macroselect..png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음
