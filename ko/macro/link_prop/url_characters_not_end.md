# URLCharactersNotEnd 속성 (LinkProp 개체)

구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의 **끝에서 문자 허용 안됨** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
s = object.URLCharactersNotEnd;
object.URLCharactersNotEnd = s;
```

### \[VBScript\]

```
s = object.URLCharactersNotEnd
object.URLCharactersNotEnd = s
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
