# NoFullPathIfNotActive 속성 (FileProp ü)

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **활성화 중이 아닐 경우 전체 경로 제외** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ = object. **NoFullPathIfNotActiveProperty**;

object. **NoFullPathIfNotActiveProperty** = _b_;

#### \[VBScript\]

_b_ = object. **NoFullPathIfNotActiveProperty**

object. **NoFullPathIfNotActiveProperty** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
