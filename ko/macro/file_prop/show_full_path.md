# ShowFullPath 속성 (FileProp ü)

구성 속성 [파일 탭](../../dlg/properties/file/index) 의전체 경로와 함께 파일 이름 보이기 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.ShowFullPath;
object.ShowFullPath = b;
```

### \[VBScript\]

```
b = object.ShowFullPath
object.ShowFullPath = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
