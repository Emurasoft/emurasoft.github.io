# MonitorInterval 속성 (FileProp 개체)

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **모니터 간격** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
n = object.MonitorInterval;
object.MonitorInterval = n;
```

### \[VBScript\]

```
n = object.MonitorInterval
object.MonitorInterval = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
