# CheckFileChanged 속성

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **기타 프로그램에 의하여 변경됨** 리스트 박스에 해당합니다.

#### \[JavaScript\]

_n_ = object. **CheckFileChanged**;

object. **CheckFileChanged** = _n_;

#### \[VBScript\]

_n_ = object. **CheckFileChanged**

object. **CheckFileChanged** = _n_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
