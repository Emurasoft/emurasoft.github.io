# InsideTag 속성 (HighlightItem 개체)

구성 속성 [**강조 표시 (1)** 탭](../../dlg/properties/highlight1/index) 의 **내부 태그 만** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = item.InsideTag;
item.InsideTag = b;
```

### \[VBScript\]

```
b = item.InsideTag
item.InsideTag = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
