# RightAll 속성 (HighlightItem ü)

구성 속성 [**강조 표시 (1)** 탭](../../dlg/properties/highlight1/index) 의 **오른쪽 모두 강조 표시** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
item. **RightAll**;

item. **RightAll** = _b_;

#### \[VBScript\]

_b_ =
item. **RightAll**

item. **RightAll** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
