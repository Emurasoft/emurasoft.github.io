# InsertSpaces 속성

[**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의 **탭을 위한 공백 삽입** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **InsertSpaces**;

object. **InsertSpaces** = _b_;

#### \[VBScript\]

_b_ =
object. **InsertSpaces**

object. **InsertSpaces** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
