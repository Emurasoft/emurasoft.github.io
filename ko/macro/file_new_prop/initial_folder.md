# InitialFolder 속성

[**새 파일 상세 정보** 대화 상자](../../dlg/properties/file/new_details/index) 의
**초기 폴더** 텍스트 박스에 해당합니다.

#### \[JavaScript\]

_s_ = object. **InitialFolder**;

object. **InitialFolder** = _s_;

#### \[VBScript\]

_s_ = object. **InitialFolder**

object. **InitialFolder** = _s_

## 버전

EmEditor Professional 버전 14.1 이상에서만 지원됩니다.
