# TextMode 속성 (DisplayItem ü)

텍스트 색상 모드를 지정합니다.

#### \[JavaScript\]

_nMode_ = item. **TextMode**;

item. **TextMode** = _nMode_;

#### \[VBScript\]

_nMode_ = item. **TextMode**

item. **TextMode** = _nMode_

## 참고

nMode 매개 변수는 다음의 값 중 하나가 되어야 합니다.

|     |     |
| --- | --- |
| 0 | 사용자 지정 |
| 1 | 기본값 |
| 2 | 투명 |
