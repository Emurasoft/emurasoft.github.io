# Style 속성 (DisplayItem ü)

개체의 스타일을 지정합니다.

#### \[JavaScript\]

_n_ = item. **Style**;

item. **Style** = _n_;

#### \[VBScript\]

_n_ = item. **Style**

item. **Style** = _n_

## 참고

nStyle 매개 변수는 다음의 값들 중 하나가 되어야 합니다.

|     |     |
| --- | --- |
| 0 | 일반 |
| 1 | 밑줄 |
| 2 | 진하게 |
| 3 | 기울임 |
