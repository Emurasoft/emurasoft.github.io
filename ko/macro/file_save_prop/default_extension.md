# DefaultExtension 속성 (FileSaveProp 개체)

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **기본 확장자** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
s = object.DefaultExtension;
object.DefaultExtension = s;
```

### \[VBScript\]

```
s = object.DefaultExtension
object.DefaultExtension = s
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
