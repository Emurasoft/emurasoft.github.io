# PromptUnicode 속성 (FileSaveProp ü)

[세부 정보 저장 대화 상자](../../dlg/properties/file/save_details/index) 의
유니코드를 ANSI로 저장할 때 확인 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.PromptUnicode;
object.PromptUnicode = b;
```

### \[VBScript\]

```
b = object.PromptUnicode
object.PromptUnicode = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
