# SameFontPrint 속성 (FontProp ü)

[글꼴 사용자 지정](../../dlg/properties/font/index) 대화 상자의글꼴 표시를 프린터 글꼴로 사용 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.SameFontPrint;
object.SameFontPrint = b;
```

### \[VBScript\]

```
b = object.SameFontPrint
object.SameFontPrint = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
