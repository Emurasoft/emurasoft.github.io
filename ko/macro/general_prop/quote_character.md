# QuoteCharacter 속성 (GeneralProp 개체)

구성 속성 [일반 탭](../../dlg/properties/general/index) 의따옴표 문자 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
str = object.QuoteCharacter;
object.QuoteCharacter = str;
```

### \[VBScript\]

```
str = object.QuoteCharacter
object.QuoteCharacter = str
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
