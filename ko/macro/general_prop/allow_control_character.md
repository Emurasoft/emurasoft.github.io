# AllowControlCharacter 속성

구성 속성 [**일반** 탭](../../dlg/properties/general/index) 의 **컨트롤 문자 삽입 허용** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **AllowControlCharacter**;

object. **AllowControlCharacter** = _b_;

#### \[VBScript\]

_b_ =
object. **AllowControlCharacter**

object. **AllowControlCharacter** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
