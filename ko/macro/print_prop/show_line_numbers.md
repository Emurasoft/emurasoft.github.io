# ShowLineNumbers 속성 (PrintProp 개체)

구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의 **줄 번호 인쇄** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.ShowLineNumbers;
object.ShowLineNumbers = b;
```

### \[VBScript\]

```
b = object.ShowLineNumbers
object.ShowLineNumbers = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
