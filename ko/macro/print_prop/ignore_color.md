# IgnoreColor 속성 (PrintProp ü)

구성 속성 [인쇄 탭](../../dlg/properties/print/index) 의색과 밑줄 무시하기 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.IgnoreColor;
object.IgnoreColor = b;
```

### \[VBScript\]

```
b = object.IgnoreColor
object.IgnoreColor = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
