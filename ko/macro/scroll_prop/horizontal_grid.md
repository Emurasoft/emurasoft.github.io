# HorizontalGrid 속성 (ScrollProp 개체)

구성 속성 [스크롤 탭](../../dlg/properties/scroll/index) 의가로 눈금 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.HorizontalGrid;
object.HorizontalGrid = b;
```

### \[VBScript\]

```
b = object.HorizontalGrid
object.HorizontalGrid = b
```

## 버전

엠에디터 프로페셔널 버전 10.00 이상에서만 지원됩니다.
