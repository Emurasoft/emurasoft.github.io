# ShowOnlyActive 속성 (ScrollProp ü)

구성 속성 [스크롤 탭](../../dlg/properties/scroll/index) 의활성 창에서만 스크롤 바 보이기 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.ShowOnlyActive;
object.ShowOnlyActive = b;
```

### \[VBScript\]

```
b = object.ShowOnlyActive
object.ShowOnlyActive = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
