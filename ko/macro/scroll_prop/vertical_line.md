# VerticalLine 속성 (ScrollProp 개체)

구성 속성 [스크롤 탭](../../dlg/properties/scroll/index) 의세로 줄 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.VerticalLine ;
object.VerticalLine  = b;
```

### \[VBScript\]

```
b = object.VerticalLine
object.VerticalLine  = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
