# prompt 메서드 (Window ü)

문자열을 입력하는 대화 상자를 표시합니다.

#### \[JavaScript\]

_strAnswer_ = **prompt**( _strMessage, strDefault_);

#### \[VBScript\]

_strAnswer_ = **prompt**( _strMessage, strDefault_ )

## 매개 변수

_strMessage_

표시할 메시지를 지정합니다.

_strDefault_

텍스트 박스에 표시될 기본 문자열을 지정합니다.

## 반환 값

확인 버튼이 선택되면 텍스트 박스에 사용자가 입력한 문자열로, 또는 취소 버튼을 선택하면 빈 문자열로 반환됩니다.

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
