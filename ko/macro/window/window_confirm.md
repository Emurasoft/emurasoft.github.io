# confirm 메서드 (Window ü)

OK 버튼과 취소 버튼을 가진 간단한 대화 상자에 메시지를 표시합니다.

#### \[JavaScript\]

_bAnswer_ = **confirm**( _strQuestion_);

#### \[VBScript\]

_bAnswer_ = **confirm**( _strQuestion_ )

## 매개 변수

_strQuestion_

표시할 메시지를 지정합니다.

## 반환 값

OK 버튼이 선택된 경우 true를 반환하거나 취소 버튼이 선택되면 false를 반환합니다.

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
