# RenameIfSameNameExists 속성

구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **같은 파일 이름이 존재할 경우 이름 바꾸기** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ = object. **RenameIfSameNameExists**;

object. **RenameIfSameNameExists** = _b_;

#### \[VBScript\]

_b_ = object. **RenameIfSameNameExists**

object. **RenameIfSameNameExists** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
