# CommentBegin 속성 (HighlightProp 개체)

구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의설명 그룹 박스 \- 시작 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
str = object.CommentBegin;
object.CommentBegin = str;
```

### \[VBScript\]

```
str = object.CommentBegin
object.CommentBegin = str
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
