# SpecialSyntax 속성

구성 속성 [**강조 표시 (2)** 탭](../../dlg/properties/highlight2/index) 의 **특수 구문 드롭 다운** 리스트 박스에 해당합니다.

#### \[JavaScript\]

_n_ = object. **SpecialSyntax**;

object. **SpecialSyntax** = _n_;

#### \[VBScript\]

_n_ = object. **SpecialSyntax**

object. **SpecialSyntax** = _n_
