# SpecialSyntax 속성 (HighlightProp 개체)

구성 속성 [**강조 표시 (2)** 탭](../../dlg/properties/highlight2/index) 의 **특수 구문 드롭 다운** 리스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
n = object.SpecialSyntax;
object.SpecialSyntax = n;
```

### \[VBScript\]

```
n = object.SpecialSyntax
object.SpecialSyntax = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
