# Enabled 속성 (HighlightProp 개체)

구성 속성 [**강조 표시 (1)** 탭](../../dlg/properties/highlight1/index) 의 **이 단어들을 강조 표시** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.Enabled;
object.Enabled = b;
```

### \[VBScript\]

```
b = object.Enabled
object.Enabled = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
