# QuoteSingle 속성 (HighlightProp ü)

구성 속성 [**강조 표시 (2)** 탭](../../dlg/properties/highlight2/index) 의 **따옴표로 닫힌 문자열 그룹 박스 \- 작은 따옴표 '...'** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **QuoteSingle**;

object. **QuoteSingle** = _b_;

#### \[VBScript\]

_b_ =
object. **QuoteSingle**

object. **QuoteSingle** = _b_
