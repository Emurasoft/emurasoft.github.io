# JavaScript 또는 VBScript 지원 ()

엠에디터는 매크로 언어로 JavaScript 또는 VBScript를 사용하므로,
HTML이나 Windows 스크립트에 익숙한 사용자들은 매크로를 기록하는데 있어서 적은 어려움이 있을수 있습니다.
스크립트 언어에 익숙하지 않은 사용자들을 위해, 엠에디터는 다른 상황에서도 쉽게 로드될 수 있도록 매크로
파일에 저장되도록 할수 있는 키 입력을 기록해 놓을 수 있습니다.
JavaScript 또는 VBScript을 사용함으로써, 코드를 쉽게 해결할 수도 있습니다.
예를 들어, JavaScript에서 다음과 같은 문장을 사용하여 오류를 해결할 수 있습니다:

try { ... } catch(e) { ... }

파일 열기 실패와 같은 오류가 발생하였을 때, try-catch 문장이 강제로 종료하도록 하는 대신
스크립트를 계속해서 실행할수 있도록 합니다.
