# CamelCase 속성 (SpellProp 개체)

구성 속성의 [**맞춤법** 탭](../../dlg/properties/spell/index) 의 **CamelCase 지원** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.CamelCase;
object.CamelCase = b;
```

### \[VBScript\]

```
b = object.CamelCase
object.CamelCase = b
```

## 버전

엠에디터 프로페셔널 버전 10.00 이상에서만 지원됩니다.
