# LineOpen 메서드 (Selection ü)

두 줄 사이에 빈 줄을 삽입합니다.

## 

### \[JavaScript\]

```
document.selection.LineOpen( [ bAbove ] );
```

### \[VBScript\]

```
document.selection.LineOpen [ bAbove ]
```

## 매개 변수

_bAbove_

선택 사항입니다.
true가 지정되면 현재 줄의 시작점에 새로운 줄을 삽입합니다.
false가 지정되면, 현재 줄의 끝에 새로운 줄을 삽입합니다.
기본 값은 false입니다.

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
