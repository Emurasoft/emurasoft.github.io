# OverwriteMode 속성

덮어쓰기 또는 삽입 모드를 나타내는 플래그를 설정하거나 검색합니다.

#### \[JavaScript\]

_bOverwrite_ = document.selection. **OverwriteMode**;

document.selection. **OverwriteMode** = _bOverwrite_;

#### \[VBScript\]

_bOverwrite_ = document.selection. **OverwriteMode**

document.selection. **OverwriteMode** = _bOverwrite_

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
