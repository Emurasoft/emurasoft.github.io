# Copy 메서드 (Selection ü)

클립보드에 선택 영역을 복사합니다.

#### \[JavaScript\]

document.selection. **Copy**( _nFlags_ );

#### \[VBScript\]

document.selection. **Copy** _nFlags_

## 매개 변수

_nFlags_

다음의 값들의 결합을 지정합니다:

|     |     |
| --- | --- |
| eeCopyUnicode | 유니코드로 복사하기 (기본값). |
| eeCopyQuotes | 따옴표안에 복사하기. |
