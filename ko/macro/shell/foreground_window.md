# ForegroundWindow 속성 (Shell 개체)

현재 전경 창을 검색합니다.

## 

### \[JavaScript\]

```
wnd = shell.ForegroundWindow;
```

### \[VBScript\]

```
wnd = shell.ForegroundWindow
```

## 예시

### \[JavaScript\]

```
wnd = shell.FouregroundWindow;
str = wnd.Caption;
alert( str );
```

### \[VBScript\]

```
wnd = shell.FouregroundWindow
str = wnd.Caption
alert str
```
