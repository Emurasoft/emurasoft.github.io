# SearchColors 속성 (DisplayProp 개체)

구성 속성 [디스플레이 탭](../../dlg/properties/display/index) 의색 검색 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
n = object.SearchColors;
object.SearchColors = n;
```

### \[VBScript\]

```
n = object.SearchColors
object.SearchColors = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
