# Path 속성 (Document 개체)

현재 문서의 경로만을 검색합니다.

## 

### \[JavaScript\]

```
strPath = document.Path;
```

### \[VBScript\]

```
strPath = document.Path
```

## 예시

### \[JavaScript\]

```
alert( document.Path );
```

### \[VBScript\]

```
alert document.Path
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
