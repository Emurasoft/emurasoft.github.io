# HighlightFind 속성 (Document 개체)

검색된 문자열을 강조 표시 할지의 여부를 결정합니다.

## 

### \[JavaScript\]

```
bHighlightFind = document.HighlightFind;
document.HighlightFind = bHighlightFind;
```

### \[VBScript\]

```
bHighlightFind = document.HighlightFind
document.HighlightFind = bHighlightFind
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
