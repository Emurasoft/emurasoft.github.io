# FullName 속성 (Document 개체)

문서의 전체 경로와 파일 이름을 검색합니다.

## 

### \[JavaScript\]

```
strName = document.FullName;
```

### \[VBScript\]

```
strName = document.FullName
```

## 예시

### \[JavaScript\]

```
alert( document.FullName );
```

### \[VBScript\]

```
alert document.FullName
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
