# HighlightTag 속성 (Document 개체)

태그를 강조 표시 할자의 여부를 결정합니다.

## 

### \[JavaScript\]

```
bHighlightTag = document.HighlightTag;
document.HighlightTag = bHighlightTag;
```

### \[VBScript\]

```
bHighlightTag = document.HighlightTag
document.HighlightTag = bHighlightTag
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
