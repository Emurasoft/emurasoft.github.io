# \#include 지침서 (Script ħ)

포함 할 파일을 지정합니다. 이 지침서는 메인 기본 코드 위 스크립트의 첫 줄에 지정되어야 합니다.

#include " _filename_"

## 매개 변수

_filename_

현재 매크로 파일 또는 전체 경로로부터 상대 경로와 포함한 파일을 지정합니다.
경로가 생략된 경우, 현재 매크로 파일이 존재하는 파일의 제목을 지정합니다.

## 예시

library.jsee 파일을 포함합니다.

#include "library.jsee"
