# \#tooltip 지침서 (Script ħ)

도구 모음 버튼에 마우스를 가져갔을 때 표시되는 툴팁 문자열을 지정합니다.
이 지침서는 메인 기본 코드 위 스크립트의 첫 줄에 지정되어야 합니다.

#tooltip = " _tooltip_"

## 매개 변수

_tooltip_

> 도구 모음 버튼에 마우스를 가져갔을 때 표시되는 툴팁 문자열을 지정합니다.
> 빈 문자열이 지정되면 툴팁은 표시되지 않습니다.

## 예시

"calculation"로 툴팁을 구성합니다.

#tooltip = "Calculation"

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
