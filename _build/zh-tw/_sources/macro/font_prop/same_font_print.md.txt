# SameFontPrint 屬性

與自訂字型對話方塊中的 [**將顯示字型作為印表機字型** 核取方塊](../../dlg/properties/font/index) 相對應。

#### \[JavaScript\]

_b_ =
object. **SameFontPrint**;

object. **SameFontPrint** = _b_;

#### \[VBScript\]

_b_ =
object. **SameFontPrint**

object. **SameFontPrint** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
