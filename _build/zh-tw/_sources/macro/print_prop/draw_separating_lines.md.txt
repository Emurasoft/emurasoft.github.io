# DrawSeparatingLines 屬性

與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **插入分隔線** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **DrawSeparatingLines**;

object. **DrawSeparatingLines** = _b_;

#### \[VBScript\]

_b_ =
object. **DrawSeparatingLines**

object. **DrawSeparatingLines** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
