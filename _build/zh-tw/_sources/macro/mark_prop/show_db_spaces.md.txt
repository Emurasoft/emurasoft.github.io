# ShowDBSpaces 扽俶

Corresponds to the **Show**
**Double-Byte Spaces** check box in the
[**Marks** tab](../../dlg/properties/marks/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
object. **ShowDBSpaces**;

object. **ShowDBSpaces** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowDBSpaces**

object. **ShowDBSpaces** = _b_
