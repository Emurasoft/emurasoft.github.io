# SetInitialFolder 屬性

與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **設置初始資料夾** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **SetInitialFolder**;

object. **SetInitialFolder** = _b_;

#### \[VBScript\]

_b_ =
object. **SetInitialFolder**

object. **SetInitialFolder** = _b_

## 版本

支持 EmEditor 14.1 或之後的版本。
