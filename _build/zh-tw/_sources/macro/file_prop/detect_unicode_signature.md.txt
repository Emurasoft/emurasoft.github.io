# DetectUnicodeSignature 屬性

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **偵測 Unicode 簽名 (BOM)** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **DetectUnicodeSignature**;

object. **DetectUnicodeSignature** = _b_;

#### \[VBScript\]

_b_ =
object. **DetectUnicodeSignature**

object. **DetectUnicodeSignature** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
