# HorizontalLine 屬性

與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **水平線** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **HorizontalLine**;

object. **HorizontalLine** = _b_;

#### \[VBScript\]

_b_ =
object. **HorizontalLine**

object. **HorizontalLine** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
