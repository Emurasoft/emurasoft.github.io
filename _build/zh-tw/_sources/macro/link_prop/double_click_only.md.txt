# DoubleClickOnly 屬性

與組態屬性中 [**連結** 頁面](../../dlg/properties/link/index) 上的 **僅允許雙擊** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **DoubleClickOnly**;

object. **DoubleClickOnly** = _b_;

#### \[VBScript\]

_b_ =
object. **DoubleClickOnly**

object. **DoubleClickOnly** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
