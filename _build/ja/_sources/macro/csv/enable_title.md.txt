# EnableTitle プロパティ

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[ツール バー ボタンのタイトル\] チェック ボックス に相当します。

#### \[JavaScript\]

_b_ = object. **EnableTitle**;

object. **EnableTitle** = _b_;

#### \[VBScript\]

_b_ = object. **EnableTitle**

object. **EnableTitle** = _b_

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
