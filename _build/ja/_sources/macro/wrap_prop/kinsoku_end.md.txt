# KinsokuEnd プロパティ

設定のプロパティの [**\[禁則文字\]** タブ](../../dlg/properties/wrap/index) の
**\[行末禁則文字\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_str_ = object. **KinsokuEnd**;

object. **KinsokuEnd** = _str_;

#### \[VBScript\]

_str_ = object. **KinsokuEnd**

object. **KinsokuEnd** = _str_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
