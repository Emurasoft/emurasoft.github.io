# Kinsoku プロパティ

設定のプロパティの [**\[禁則文字\]** タブ](../../dlg/properties/wrap/index) の
**\[禁則処理を行う\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ = object. **Kinsoku**;

object. **Kinsoku** = _b_;

#### \[VBScript\]

_str_ = object. **Kinsoku**

object. **Kinsoku** = _str_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
