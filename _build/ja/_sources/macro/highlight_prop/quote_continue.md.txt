# QuoteContinue プロパティ

設定のプロパティの [**\[強調(2)\]** タブ](../../dlg/properties/highlight2/index) の \[引用符に囲まれた文字列\] グループ \- \[次の行へ続く\] チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **QuoteContinue**;

object. **QuoteContinue** = _b_;

#### \[VBScript\]

_b_ =
object. **QuoteContinue**

object. **QuoteContinue** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
