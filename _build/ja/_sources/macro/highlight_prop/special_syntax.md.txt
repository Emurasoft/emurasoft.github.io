# SpecialSyntax プロパティ

設定のプロパティの [**\[強調(2)\]** タブ](../../dlg/properties/highlight2/index) の **\[特別な構文\]** ドロップダウン リスト ボックスに相当します。

#### \[JavaScript\]

_n_ = object. **SpecialSyntax**;

object. **SpecialSyntax** = _n_;

#### \[VBScript\]

_n_ = object. **SpecialSyntax**

object. **SpecialSyntax** = _n_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
