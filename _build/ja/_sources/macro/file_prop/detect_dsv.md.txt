# DetectDSV プロパティ

設定のプロパティの [**\[ファイル\]** タブ](../../dlg/properties/file/index) の **\[DSV (ユーザー定義区切り) を検出\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ = object. **DetectDSV**;

object. **DetectDSV** = _b_;

#### \[VBScript\]

_b_ = object. **DetectDSV**

object. **DetectDSV** = _b_

## バージョン

EmEditor Professional Version 9.00 以上で利用できます。
