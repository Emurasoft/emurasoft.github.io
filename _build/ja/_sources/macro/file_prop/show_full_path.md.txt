# ShowFullPath プロパティ

設定のプロパティの [**\[ファイル\]** タブ](../../dlg/properties/file/index) の **\[ファイル名をフルパスで表示\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **ShowFullPath**;

object. **ShowFullPath** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowFullPath**

object. **ShowFullPath** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
