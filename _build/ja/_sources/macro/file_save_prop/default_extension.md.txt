# DefaultExtension プロパティ

設定のプロパティの [**\[ファイル\]** タブ](../../dlg/properties/file/index) の **\[既定の拡張子\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_s_ = object. **DefaultExtension**;

object. **DefaultExtension** = _s_;

#### \[VBScript\]

_s_ = object. **DefaultExtension**

object. **DefaultExtension** = _s_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
