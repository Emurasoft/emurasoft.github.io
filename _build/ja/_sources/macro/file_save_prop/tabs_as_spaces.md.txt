# TabsAsSpaces プロパティ

[**\[保存時の詳細\]** ダイアログ ボックス](../../dlg/properties/file/save_details/index) の
**\[保存時、タブをスペースに変換\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **TabsAsSpaces**;

object. **TabsAsSpaces** = _b_;

#### \[VBScript\]

_b_ =
object. **TabsAsSpaces**

object. **TabsAsSpaces** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
