# IndentColumns プロパティ

[**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の **\[インデントの桁数\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_n_ =
object. **IndentColumns**;

object. **IndentColumns** = _n_;

#### \[VBScript\]

_n_ =
object. **IndentColumns**

object. **IndentColumns** = _n_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
