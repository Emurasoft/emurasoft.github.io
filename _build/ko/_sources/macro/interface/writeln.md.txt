# writeln 메서드

지원되는 플러그 인에 문자열과 새로운 줄을 전달합니다.

#### \[JavaScript\]

Interface. **writeln**( _str_ );

#### \[VBScript\]

Interface. **writeln** _str_

## 매개 변수

_str_

전달될 문자열을 지정합니다.

## 예시

#### \[JavaScript\]

Interface. **writeln**( "Hello" );

#### \[VBScript\]

Interface. **writeln** "Hello"

## 버전

엠에디터 프로페셔널 버전 9.00 이상에서만 지원됩니다.
