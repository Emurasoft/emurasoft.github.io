# CharacterSpace 속성

구성 속성 [**디스플레이** 탭](../../dlg/properties/display/index) 의 **문자 공백** 텍스트 박스에 해당합니다.

#### \[JavaScript\]

_n_ =
object. **CharacterSpace**;

object. **CharacterSpace** = _n_;

#### \[VBScript\]

_n_ =
object. **CharacterSpace**

object. **CharacterSpace** = _n_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
