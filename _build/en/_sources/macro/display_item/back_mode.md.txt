# BackMode 屬性

指定背景顏色模式。

#### \[JavaScript\]

_nMode_ = item. **BackMode**;

item. **BackMode** = _nMode_;

#### \[VBScript\]

_nMode_ = item. **BackMode**

item. **BackMode** = _nMode_

## 備注

nMode 參數會是下列值之一。

|     |     |
| --- | --- |
| 0 | 自訂 |
| 1 | 預設 |
| 2 | 透明 |

## 版本

支持 EmEditor 7.00 或之後的版本。
