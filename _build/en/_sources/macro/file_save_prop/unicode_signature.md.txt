# UnicodeSignature 屬性

與 [**添加一個 Unicode 簽名 (BOM)** 核取方塊](../../dlg/properties/file/save_details/index) 相對應。

#### \[JavaScript\]

_b_ =
object. **UnicodeSignature**;

object. **UnicodeSignature** = _b_;

#### \[VBScript\]

_b_ =
object. **UnicodeSignature**

object. **UnicodeSignature** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
