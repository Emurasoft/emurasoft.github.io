# confirm 方法

在一個簡單的對話方塊內顯示一條消息以及「OK」和「Cancel」按鈕。

#### \[JavaScript\]

_bAnswer_ = **confirm**( _strQuestion, strMessage2_);

#### \[VBScript\]

_bAnswer_ = **confirm**( _strQuestion, strMessage2_ )

## 參數

_strQuestion_

指定要顯示的消息。

_strMessage2_

可選項。指定要在第一條消息下方顯示的額外的消息。

## 返回值

返回 true 如果選擇「OK」按鈕，或者返回 false 如果選擇「Cancel」按鈕。

## 版本

支持 EmEditor 4.00 或之後的版本。
