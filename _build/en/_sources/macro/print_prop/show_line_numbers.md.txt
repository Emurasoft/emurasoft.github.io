# ShowLineNumbers 屬性

與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **列印行號** 核取方塊相對應。

#### \[JavaScript\]

_b_ = object. **ShowLineNumbers**;

object. **ShowLineNumbers** = _b_;

#### \[VBScript\]

_b_ = object. **ShowLineNumbers**

object. **ShowLineNumbers** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
