# AllowDelimiter 属性

与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **允许分隔符在引号内** 复选框对应。

#### \[JavaScript\]

_b_ = object. **AllowDelimiter**;

object. **AllowDelimiter** = _b_;

#### \[VBScript\]

_b_ = object. **AllowDelimiter**

object. **AllowDelimiter** = _b_

## 版本

支持 EmEditor 19.4 或之后的版本。
