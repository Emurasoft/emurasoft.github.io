# TemplateFile 属性

与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **模板文件** 文本框相对应。

#### \[JavaScript\]

_s_ = object. **TemplateFile**;

object. **TemplateFile** = _s_;

#### \[VBScript\]

_s_ = object. **TemplateFile**

object. **TemplateFile** = _s_

## 版本

支持 EmEditor 7.00 或之后的版本。
