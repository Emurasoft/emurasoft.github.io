# SetInitialFolder 属性

Corresponds to the **Set Initial Folder** check box in the
[**New File Details** dialog box](../../dlg/properties/file/new_details/index).

#### \[JavaScript\]

_b_ =
object. **SetInitialFolder**;

object. **SetInitialFolder** = _b_;

#### \[VBScript\]

_b_ =
object. **SetInitialFolder**

object. **SetInitialFolder** = _b_

## 版本

支持 EmEditor 14.1 或之后的版本。
