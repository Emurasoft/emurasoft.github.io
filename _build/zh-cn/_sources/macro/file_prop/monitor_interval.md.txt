# MonitorInterval 属性

与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **监控时间间隔** 文本框相对应。

#### \[JavaScript\]

_n_ = object. **MonitorInterval**;

object. **MonitorInterval** = _n_;

#### \[VBScript\]

_n_ = object. **MonitorInterval**

object. **MonitorInterval** = _n_

## 版本

支持 EmEditor 7.00 或之后的版本。
