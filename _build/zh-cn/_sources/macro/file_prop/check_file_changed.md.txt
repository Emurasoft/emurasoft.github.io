# CheckFileChanged 属性

与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **被其他程序更改时** 下拉列表框相对应。

#### \[JavaScript\]

_n_ = object. **CheckFileChanged**;

object. **CheckFileChanged** = _n_;

#### \[VBScript\]

_n_ = object. **CheckFileChanged**

object. **CheckFileChanged** = _n_

## 版本

支持 EmEditor 7.00 或之后的版本。
