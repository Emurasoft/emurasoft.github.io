# BackMode 属性

指定背景颜色模式。

#### \[JavaScript\]

_nMode_ = item. **BackMode**;

item. **BackMode** = _nMode_;

#### \[VBScript\]

_nMode_ = item. **BackMode**

item. **BackMode** = _nMode_

## 备注

nMode 参数会是下列值之一。

|     |     |
| --- | --- |
| 0 | 自定义 |
| 1 | 默认 |
| 2 | 透明 |

## 版本

支持 EmEditor 7.00 或之后的版本。
