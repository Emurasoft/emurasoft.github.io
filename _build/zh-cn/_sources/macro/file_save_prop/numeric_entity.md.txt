# NumericEntity 属性

与 [**保存详细信息** 对话框](../../dlg/properties/file/save_details/index) 中的 **把 Unicode 保存为 HTML/XML 字符引用** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **NumericEntity**;

object. **NumericEntity** = _b_;

#### \[VBScript\]

_b_ =
object. **NumericEntity**

object. **NumericEntity** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
