# DoubleClickOnly 属性

与配置属性中 [**链接** 页面](../../dlg/properties/link/index) 上的 **仅允许双击** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **DoubleClickOnly**;

object. **DoubleClickOnly** = _b_;

#### \[VBScript\]

_b_ =
object. **DoubleClickOnly**

object. **DoubleClickOnly** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
