# ShowLineNumbers 属性

与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **打印行号** 复选框相对应。

#### \[JavaScript\]

_b_ = object. **ShowLineNumbers**;

object. **ShowLineNumbers** = _b_;

#### \[VBScript\]

_b_ = object. **ShowLineNumbers**

object. **ShowLineNumbers** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
