# SaveToFolder 属性

与配置属性中 [**备份** 页面](../../dlg/properties/backup/index) 上的 **保存备份文件到备份文件夹** 复选框对应。

#### \[JavaScript\]

_b_ = object. **SaveToFolder**;

object. **SaveToFolder** = _b_;

#### \[VBScript\]

_b_ = object. **SaveToFolder**

object. **SaveToFolder** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
