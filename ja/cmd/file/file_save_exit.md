# \[保存して閉じる\] コマンド

### 概要

> ファイルを保存して閉じます。

### 説明

> 作業中の文書の変更を保存して閉じます。

### 実行方法

- 既定のメニュー: \[ファイル\] \- \[保存して閉じる\]
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[閉じる\] \- \[保存して閉じる\]
- ツール バー: ![](../../images/filesaveexit.gif)
- ステータス バー: なし
- 既定のショートカット: Ctrl+E

### プラグイン コマンド ID

- EEID\_FILE\_SAVE\_EXIT (4116)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4116);

#### \[VBScript\]

> editor.ExecuteCommandByID 4116
