# \[スニペットの挿入\] コマンド

## 概要

指定した最近使ったファイルを挿入します (複数項目)。

## 説明

このコマンドは通常、複数のメニュー項目から成り立っています。このコマンドを実行すると、カーソル位置に指定したスニペットを挿入します。

## 実行方法

- 既定のメニュー: \[挿入\] \- (スニペットの挿入)
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[スニペット\] \- (スニペットの挿入)
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_INSERT_SNIPPET1 から EEID_INSERT_SNIPPET1 + 1023 まで (30208 から 30208 + 1023 まで)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(30208 + i); // i は 0 から 1023 までの整数
```

### \[VBScript\]

```
editor.ExecuteCommandByID 30208 + i   ' i は 0 から 1023 までの整数
```
