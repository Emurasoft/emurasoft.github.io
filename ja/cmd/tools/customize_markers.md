# \[マーカーのカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[マーカー\] ページを開きます。

## 説明

\[カスタマイズ\] ダイアログ ボックスの \[マーカー\] ページを開きます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[マーカー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
