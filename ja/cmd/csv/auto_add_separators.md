# \[自動的にセパレーターを追加\] コマンド

## 概要

ノーマル モード時、固定幅列への変換の準備のため、セパレーターを自動的に検出して追加します。

## 説明

ノーマル モード時、固定幅列への変換の準備のため、セパレーターを自動的に検出して追加します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セパレーター\] - \[自動的にセパレーターを追加\]
- ツール バー: ![](../../images/columns_separators..png) \+ \[自動的にセパレーターを追加\]
- ステータス バー: なし
- 既定のショートカット: なし

```
EEID_AUTO_ADD_SEPARATORS (3885)
```
