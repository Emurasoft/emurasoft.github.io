# \[列を追加\] コマンド

## 概要

CSV文書の最終列の右側に新規列を挿入します。

## 説明

CSV文書の最終列の右側に新規列を挿入します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列\] - \[列を追加\]
- ツール バー: ![](../../images/columns_separators.png) \+ \[列を追加\]
- ステータス バー: なし
- 既定のショートカット: なし
