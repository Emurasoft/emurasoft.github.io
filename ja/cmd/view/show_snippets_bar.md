# \[スニペット ツール バー\] コマンド

## 概要

スニペット ツール バーの表示/非表示を切り替えます。

## 説明

スニペット ツール バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[スニペット\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[スニペット\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_SNIPPETS_BAR (23287)
```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23287);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23287
```
