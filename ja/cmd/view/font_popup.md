# \[フォントの設定 (ポップアップ メニュー)\] コマンド

## 概要

ポップアップ メニューを表示してフォントを設定します。

## 説明

このコマンドを実行すると、ポップアップ メニューを表示します。ポップアップ メニューでは、 [\[最近使ったフォント\] \
コマンド](mru_font1)、フォント分類の選択、 [\[フォントの設定\] コマンド](font) を選択することができます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[フォント\]  \- \[フォントの設定 (ポップアップ メニュー)\]
- ツール バー: ![](../../images/fontpopup.gif) (右側の矢印の部分)
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_FONT_POPUP (4282)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4282);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4282
```
