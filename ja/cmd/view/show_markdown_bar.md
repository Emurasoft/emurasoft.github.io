# \[Markdown/HTML ツール バー\] コマンド

## 概要

Markdown/HTML ツール バーの表示/非表示を切り替えます。

## 説明

Markdown/HTML ツール バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[Markdown\/HTML]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[Markdown/HTML\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_MARKDOWN_BAR (23274)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23274);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23274
```
