# \[行番号\] コマンド

## 概要

行番号の表示/非表示を切り替えます。

## 説明

行番号の表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[行番号\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[行番号\]
- ツール バー: ![](../../images/line_number24x16..png)
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_VIEW_LINE_NUMBERS (4530)
```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4530);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4530
```
