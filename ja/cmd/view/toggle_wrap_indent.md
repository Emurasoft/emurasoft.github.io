# \[折り返しインデント\] コマンド

## 概要

折り返しインデントを有効にします。

## 説明

折り返しインデントを有効にします。

## 実行方法

- 既定のメニュー: \[表示\] \- \[タブ/インデント\] \- \[折り返しインデント\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[タブ/インデント\] \- \[折り返しインデント\]
- ツール バー: ![](../../images/wrap_indent24x16.png)
- ステータス バー: なし
- 既定のショートカット: なし

```
EEID_TOGGLE_WRAP_INDENT (4542)
```

## マクロ

editor.ExecuteCommandByID(4542);
