# \[重複行の削除\] コマンド

## 概要

選択範囲または文書全体から重複行を削除します。

## 説明

選択範囲または文書全体から重複行を削除します。

## 実行方法

- 既定のメニュー: \[並べ替え\] \- \[重複行の削除\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[重複行の削除\]
- ツール バー: ![](../../images/deleteduplicatelines..png)
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_DELETE_DUPLICATE (4564)
```

## マクロ

### \[JavaScript\]

```
document.DeleteDuplicates("",0);
```

### \[VBScript\]

```
document.DeleteDuplicates "",0
```
