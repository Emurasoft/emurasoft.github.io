# \[AI ツール バー\] コマンド

## 概要

AI ツール バーの表示/非表示を切り替えます。

## 説明

AI ツール バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[AI\] \- \[AIツール バー\]
- [すべてのコマンド](../../glossary/allcommands): \[AI\] \- \[AIツール バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_AI_BAR (23253)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23253);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23253
```
