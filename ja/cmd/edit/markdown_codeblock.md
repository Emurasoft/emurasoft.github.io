# \[コード ブロック\] コマンド

## 概要

現在行をコード ブロックに設定します。

## 説明

現在行をコード ブロックに設定します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[Markdown\] \- \[コード ブロック\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[Markdown\] \- \[コード ブロック\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MARKDOWN_CODEBLOCK (23264)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23264);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23264
```
