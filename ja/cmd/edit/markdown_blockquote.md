# \[ブロック引用\] コマンド

## 概要

現在行をブロック引用に設定します。

## 説明

現在行をブロック引用に設定します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[Markdown/HTML\] \- \[ブロック引用\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[Markdown/HTML\] \- \[ブロック引用\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MARKDOWN_BLOCKQUOTE (23263)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23263);
```

```
editor.ExecuteCommandByID 23263
