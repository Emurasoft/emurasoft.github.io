# \[CSVコンバーター\] コマンド

### 概要

> CSVコンバーターの表示/非表示を切り替えます。

### 説明

> CSVコンバーターの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: \[CSV\] - \[CSVコンバーター\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[CSVコンバーター\]
- ツール バー: ![](../../images/convert_to.gif)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CSV\_COVERTER (3891)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3891);

#### \[VBScript\]

> editor.ExecuteCommandByID 3891
