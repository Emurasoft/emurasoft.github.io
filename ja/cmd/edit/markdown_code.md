# \[Code\] コマンド

## 概要

コードのスタイルのオン/オフを切り替えます。

## 説明

コードのスタイルのオン/オフを切り替えます。

## 実行方法

- 既定のメニュー: \[編集\] \- \[Markdown/HTML\] \- \[Code\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[Markdown/HTML\] \- \[Code\]
- ツール バー: ![](../../images/markdown_code.png)
- ステータス バー: なし
- 既定のショートカット: なし
