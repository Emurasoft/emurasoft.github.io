# \[CSVモード\] コマンド

### 概要

> CSV (カンマ区切り) モードを選択します。

### 説明

> CSV (カンマ区切り) モードを選択します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): なし
- ツール バー: ![](../../images/csv.gif)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MODE\_CSV (4459)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4459);

#### \[VBScript\]

> editor.ExecuteCommandByID 4459
