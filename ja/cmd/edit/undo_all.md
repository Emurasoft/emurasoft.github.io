# \[すべて元に戻す\] コマンド

### 概要

> 以前のすべての操作を元に戻します。

### 説明

> 以前のすべての操作を元に戻します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[元に戻す\] \- \[すべて元に戻す\]
- ツール バー: ![](../../images/editundo.gif) (矢印の部分)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_UNDO\_ALL (3980)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3980);

#### \[VBScript\]

> editor.ExecuteCommandByID 3980
