# \[ハイパーリンク\] コマンド

## 概要

ハイパーリンクを挿入します。

## 説明

ハイパーリンクを挿入します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[Markdown/HTML\] \- \[ハイパーリンク\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[Markdown/HTML\] \- \[ハイパーリンク\]
- ツール バー: ![](../../images/hyperlink.png)
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MARKDOWN_HYPERLINK (23271)
```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23271);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23271
```
