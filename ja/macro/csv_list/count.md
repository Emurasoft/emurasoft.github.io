# Count プロパティ (CsvList コレクション)

Csv オブジェクトの数を取得します。

## 

### \[JavaScript\]

```
nCount = editor.CsvList.Count;
```

### \[VBScript\]

```
nCount = editor.CsvList.Count
```

## 例

### \[JavaScript\]

```
alert( "Number of Csv objects: " + editor.CsvList.Count );
```

### \[VBScript\]

```
alert "Number of Csv objects: " & editor.CsvList.Count
```

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
