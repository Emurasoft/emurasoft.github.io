# ShowIndentGuides プロパティ

設定のプロパティの \[記号\] タブの \[インデント ガイドを表示\] に相当します。

#### \[JavaScript\]

_b_ =
object. **ShowIndentGuides**;

object. **ShowIndentGuides** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowIndentGuides**;

object. **ShowIndentGuides** = _b_;

## バージョン

EmEditor Professional Version 8.00 以上で利用できます。
