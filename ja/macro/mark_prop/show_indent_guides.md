# ShowIndentGuides プロパティ (MarkProp オブジェクト)

設定のプロパティの \[記号\] タブの \[インデント ガイドを表示\] に相当します。

## 

### \[JavaScript\]

```
b = object.ShowIndentGuides;
object.ShowIndentGuides = b;
```

### \[VBScript\]

```
b = object.ShowIndentGuides;
object.ShowIndentGuides = b;
```

## バージョン

EmEditor Professional Version 8.00 以上で利用できます。
