# TabColumns プロパティ

[**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の
**\[タブの桁数\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_n_ =
object. **TabColumns**;

object. **TabColumns** = _n_;

#### \[VBScript\]

_n_ =
object. **TabColumns**

object. **TabColumns** = _n_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
