# Header プロパティ

設定のプロパティの [**\[印刷\]** タブ](../../dlg/properties/print/index) の
**\[ヘッダ\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_s_ = object. **Header**;

object. **Header** = _s_;

#### \[VBScript\]

_s_ = object. **Header**

object. **Header** = _s_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
