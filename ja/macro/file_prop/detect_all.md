# DetectAll プロパティ (FileProp オブジェクト)

設定のプロパティの **[\[ファイル\]](../../dlg/properties/file/index)** [タブ](../../dlg/properties/file/index) の
**\[すべて自動検出\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.DetectAll;
object.DetectAll = b;
```

### \[VBScript\]

```
b = object.DetectAll
object.DetectAll = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
