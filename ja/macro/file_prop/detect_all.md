# DetectAll プロパティ

設定のプロパティの **[\[ファイル\]](../../dlg/properties/file/index)** [タブ](../../dlg/properties/file/index) の
**\[すべて自動検出\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ = object. **DetectAll**;

object. **DetectAll** = _b_;

#### \[VBScript\]

_b_ = object. **DetectAll**

object. **DetectAll** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
