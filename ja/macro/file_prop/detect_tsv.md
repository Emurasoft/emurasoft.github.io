# DetectTSV プロパティ (FileProp オブジェクト)

設定のプロパティの [**\[ファイル\]** タブ](../../dlg/properties/file/index) の
**\[TSV（タブ区切り）を検出\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.DetectTSV;
object.DetectTSV = b;
```

### \[VBScript\]

```
b = object.DetectTSV
object.DetectTSV = b
```

## バージョン

EmEditor Professional Version 9.00 以上で利用できます。
