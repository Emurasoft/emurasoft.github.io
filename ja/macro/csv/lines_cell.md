# LinesCell プロパティ

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[セル内の最大改行数\] テキスト ボックス に相当します。

#### \[JavaScript\]

_n_ = object. **LinesCell**;

object. **LinesCell** = _n_;

#### \[VBScript\]

_n_ = object. **LinesCell**

object. **LinesCell** = _n_

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
