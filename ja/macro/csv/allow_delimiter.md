# AllowDelimiter プロパティ

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[引用符内の区切り文字を許可\] チェック ボックス に相当します。

#### \[JavaScript\]

_b_ = object. **AllowDelimiter**;

object. **AllowDelimiter** = _b_;

#### \[VBScript\]

_b_ = object. **AllowDelimiter**

object. **AllowDelimiter** = _b_

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
