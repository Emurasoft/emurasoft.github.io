# RowHeading プロパティ (Csv オブジェクト)

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[行ヘディング\] テキスト ボックス に相当します。

## 

### \[JavaScript\]

```
n = object.RowHeading;
object.RowHeading = n;
```

### \[VBScript\]

```
n = object.RowHeading
object.RowHeading = n
```

## バージョン

EmEditor Professional Version 22.2 以上で利用できます。
