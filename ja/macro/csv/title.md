# Title プロパティ (Csv IuWFNg)

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[ツール バー ボタンのタイトル\] テキスト ボックス に相当します。

#### \[JavaScript\]

_s_ = object. **Title**;

object. **Title** = _s_;

#### \[VBScript\]

_s_ = object. **Title**

object. **Title** = _s_

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
