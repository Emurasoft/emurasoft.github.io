# TagEnd プロパティ

設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の **\[タグ終了\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_str_ = object. **TagEnd**;

object. **TagEnd** = _str_;

#### \[VBScript\]

_str_ = object. **TagEnd**

object. **TagEnd** = _str_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
