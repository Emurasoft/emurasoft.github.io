# InsideTag プロパティ (HighlightItem オブジェクト)

設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の
**\[タグの内側のみ\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = item.InsideTag;
item.InsideTag = b;
```

### \[VBScript\]

```
b = item.InsideTag
item.InsideTag = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
