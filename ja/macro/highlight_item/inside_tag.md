# InsideTag プロパティ

設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の
**\[タグの内側のみ\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
item. **InsideTag**;

item. **InsideTag** = _b_;

#### \[VBScript\]

_b_ =
item. **InsideTag**

item. **InsideTag** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
