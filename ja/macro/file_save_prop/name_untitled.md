# NameUntitled プロパティ

[**\[保存時の詳細\]** ダイアログ ボックス](../../dlg/properties/file/save_details/index) の **\[自動的に無題の文書に名前を付ける\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **NameUntitled**;

object. **NameUntitled** = _b_;

#### \[VBScript\]

_b_ =
object. **NameUntitled**

object. **NameUntitled** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
