# MarginQuote プロパティ

設定のプロパティの [**\[基本\]** タブ](../../dlg/properties/general/index) の
**\[引用行の文字数\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **MarginQuote**;

object. **MarginQuote** = _b_;

#### \[VBScript\]

_b_ =
object. **MarginQuote**

object. **MarginQuote** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
