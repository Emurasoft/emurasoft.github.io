# LineNumbersZeroBase プロパティ (GeneralProp オブジェクト)

設定のプロパティの [\[基本\] ページ](../../dlg/properties/general/index) の \[0から始まる行、桁の番号を表示\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.LineNumbersZeroBase;
object.LineNumbersZeroBase = b;
```

### \[VBScript\]

```
b = object.LineNumbersZeroBase
object.LineNumbersZeroBase = b
```

## バージョン

EmEditor Professional Version 26.1 以上で利用できます。
