# Begin プロパティ

検索したいテキストの開始位置のインデックスを論理文字単位で指定します。テキストの最後から数えて End で指定する場合には 0 を指定します。このフィールドを有効にするには Column パラメータに -1 を指定する必要があります。

#### \[JavaScript\]

_n_ =
item. **Begin**;

item. **Begin** = _n_;

#### \[VBScript\]

_n_ =
item. **Begin**

item. **Begin** = _n_

## バージョン

EmEditor Professional Version 16.0 以上で利用できます。
