# HorizontalLine プロパティ

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の **\[水平罫線\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **HorizontalLine**;

object. **HorizontalLine** = _b_;

#### \[VBScript\]

_b_ =
object. **HorizontalLine**

object. **HorizontalLine** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
