# VerticalPage プロパティ

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の
**\[1頁の垂直スクロールを常に有効\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **VerticalPage** ;

object. **VerticalPage**  = _b_;

#### \[VBScript\]

_b_ =
object. **VerticalPage**

object. **VerticalPage**  = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
