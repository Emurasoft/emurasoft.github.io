# Smooth プロパティ (ScrollProp オブジェクト)

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の **\[スムーズ スクロール\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Smooth;
object.Smooth = b;
```

### \[VBScript\]

```
b = object.Smooth
object.Smooth = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
