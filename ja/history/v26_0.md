# Version 26.0

2026年 1月 28日

## 一般の新機能

- 「お気に入り」機能に、ファイルだけでなくフォルダーを追加できるようになりました。
- 文字や単語が選択されていない場合でも、Markdown の **太字**／*斜体*／`コード` コマンドを有効にしました。
- [元に戻す/やり直しの履歴] ダイアログはカスタム バーになりました。
- 

## 新しい設定

- [カスタマイズ] ダイアログ ボックスの [\[お気に入り\] ページ](../dlg/customize/favorites/index) に [フォルダーを追加] ボタンを追加しました。
- [カスタマイズ] ダイアログ ボックスの [\[AIオプション\] ページ](../dlg/customize/ai/index) の [優先モデル] ドロップダウン リストに `gpt-5.2` と `gpt-5.2.codex` を追加しました。
