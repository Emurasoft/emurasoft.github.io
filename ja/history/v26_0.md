# Version 26.0

2026年 2月 4日

## 一般の新機能

- [お気に入り] に、フォルダーとファイルの両方を追加できるようになりました。 {{profree}}
- Markdown の **太字**/*斜体*/`コード` コマンドが、テキスト未選択の場合でも動作するようになりました。 {{pro}}
- [元に戻す/やり直し履歴] ダイアログを、カスタムバーに置き換えました。 {{profree}}
- [特殊文字の挿入] ダイアログを、カスタムバーに置き換えました。フォントサイズの拡大、カラーフォント対応、Unicode 名およびブロックによるフィルタリングなどの改善を含みます。 {{profree}}
- [文字コード値] ダイアログを、フォントサイズの拡大とカラー対応により改善しました。 {{profree}}
- 不自然な英語、ドイツ語、日本語の UI 文字列を改善しました。 {{profree}}
- v25.4 をもって、ロシア語 UI の提供を終了しました。 {{profree}}
- 改行を含まない巨大ファイルのオープンを含む様々な動作を高速化しました。{{profree}}

### テスト結果:

|  | v25.4 | v26.0 | v25.4 と比較 |
| --- | --- | --- | --- |
| 改行を含まない巨大ファイルを開く \[\*a\] | 1.461 秒 | 0.666 秒 | 2.20 倍に高速化 |
| 検索 (すべて選択)  [\*b\] | 43.3 秒 | 0.065 秒 | 666 倍に高速化 |
| すべて置換 [\*c\] | 44.1 秒 | 0.359 秒 | 123 倍に高速化 |

- \*a: ファイルを開く。
- \*b: "abc" をすべて選択。
- \*c: "abc" を "###" にすべて置換。
- \*すべて: Wikipedia ダンプ ファイル (UTF-8、1.98 GB、改行を含まない)。すべて既定オプション、Windows 11、Ryzen 9 7950X、96 GB RAM。

## 新しい設定

- [カスタマイズ] ダイアログ ボックスの [\[お気に入り\] ページ](../dlg/customize/favorites/index) に [フォルダーを追加] ボタンを追加しました。 {{profree}}
- [カスタマイズ] ダイアログ ボックスの [\[AIオプション\] ページ](../dlg/customize/ai/index) の [優先モデル] ドロップダウン リストに `gpt-5.2` と `gpt-5.2.codex` を追加しました。 {{pro}}

## プラグインの新機能 {{pro}}

- [AIとチャット] プラグインでチャットにファイルをアップロードできる機能を追加しました。
- [AIとチャット] プラグインにチャット メッセージを検索できる検索機能を追加しました。
- [コミット リスト] の変更サイドバーのパフォーマンスを改善しました。

### 注

- \[P\] EmEditor Professional のみ
- \[PF\] EmEditor Professional と EmEditor Free
