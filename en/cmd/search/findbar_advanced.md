# Advanced (Find toolbar) command

## Summary

Displays the Advanced dialog box for the Find toolbar.

## Description

Displays the Advanced dialog box for the Find toolbar.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Search**
\> **Find Toolbar** \> **Advanced**
- Toolbar: ![](../../images/commonsettings.png)
(Find toolbar)
- Status Bar: None
- Default Shortcut Key: None
