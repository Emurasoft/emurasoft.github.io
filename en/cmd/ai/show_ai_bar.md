# AI Toolbar Command

## Summary

Shows or hides the AI toolbar.

## Description

Shows or hides the AI toolbar.

## How to Run

- Default Menu: **AI** \> **AI Toolbar**
- [All Commands](../tools/all_commands): **AI** \> **AI Toolbar**
- Tool Bar: None
- Status Bar: None
- Default Shortcut Key: None
