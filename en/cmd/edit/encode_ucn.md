# Unicode to Universal Character Names command

## Summary

Encodes the selected text to Universal Character Names.

## Description

Encodes the selected text to Universal Character Names.

- [All Commands](../tools/all_commands):Convert \>Encode/Decode \>Unicode to Universal Character Names
