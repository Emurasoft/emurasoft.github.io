# Version 26.0  
February 4, 2026  

## New General Features

- You can now add both folders and files to **Favorites**. {{profree}}
- The Markdown **Bold**/*Italic*/`Code` command now works even when no text is selected. {{pro}}
- The **Undo/Redo History** dialog has been replaced with a custom bar. {{profree}}  
- The **Insert Special Character** dialog has been replaced with a custom bar, with improvements including a larger, color font and filtering by Unicode name and block. {{profree}} 
- The **Character Code Value** dialog has been improved with a larger font and color support. {{profree}} 
- Improved awkward English, German, and Japanese UI text. {{profree}} 
- Discontinued the Russian UI in v25.4. {{profree}}
- Improved the speed of various operations including **Opening a large file containing no newlines (one-line file)**. {{profree}}

### Test results:

|  | v25.4 | v26.0 | Comparison to v25.3 |
| --- | --- | --- | --- |
| Open a large file without newlines [\*a\] | 1.426 seconds | 0.619 seconds | 2.30 times faster |
| Find (Select All)  [\*b\] | 43.3 seconds | 0.065 seconds | 666 times faster |
| Replace All  [\*c\] | 44.1 seconds | 0.359 seconds | 123 times faster |

- \*a: Open a file.
- \*b: Select all "abc".
- \*c: Replace all "abc" with "###".
- \*All: A Wikipedia dump file (UTF-8, 1.98 GB, no newline), All default options, Windows 11, Ryzen 9 7950X, 96 GB RAM.
