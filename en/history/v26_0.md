# Version 26.0  
January 28, 2026  

## New General Features

- You can now add both folders and files to **Favorites**.  
- The Markdown **Bold**/*Italic*/`Code` command now works even when no text is selected.  
- The **Undo/Redo History** dialog has been replaced with a custom bar.  
- The **Insert Special Character** dialog has been replaced with a custom bar, with improvements including a larger, color font and filtering by Unicode name and block.  
- The **Character Code Value** dialog has been improved with a larger font and color support.  
- Improved awkward English, German, and Japanese UI text.  
- Discontinued the Russian UI.

## New Options

- Added the **Add Folder** button to the [**Favorites** page](../dlg/customize/favorites/index) of the **Customize** dialog box.
- Added `gpt-5.2` and `gpt-5.2.codex` to the **Preferred Model** drop-down list in the [**AI Options** page](../dlg/customize/ai/index) of the **Customize** dialog box.
