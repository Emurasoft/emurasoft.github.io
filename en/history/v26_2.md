# Version 26.2

June 29, 2026

## General New Features

- The **AI-Assisted Writing** feature now supports additional AI providers, including **Anthropic**, **DeepSeek**, **Google**, and **LM Studio/OpenAI-compatible** providers, in addition to the existing **OpenAI** support. This feature requires the **ChatAI** plug-in. {{pro}}
- **AI Prompts** now allow you to select a second provider/model, so you can send the same prompt to two AIs at the same time. You can also compare the responses from the two AIs side by side in a split view. This feature requires the **ChatAI** plug-in. {{pro}}
- Improved performance, including **horizontal scrolling**, for large files with no newline characters (single-line files). {{profree}}

## New Options

- Added a **Provider** drop-down list box to the [**AI Options** page](../dlg/customize/ai/index) of the **Customize** dialog box. {{pro}}
- Removed the **Use 'OPENAI_API_KEY' environment variable** checkbox and **OpenAI API key** text box from the [**AI Options** page](../dlg/customize/ai/index) of the **Customize** dialog box. {{pro}}
- Added the **Set Second Provider/Model** checkbox, the **Provider 2** checkbox and drop-down list box, and the **Model 2** checkbox and drop-down list box to the [**AI Prompts** page](../dlg/customize/ai_list/index) of the **Customize** dialog box. {{pro}}
- Removed the **HD** and **Vivid** checkboxes from the [**AI Prompts** page](../dlg/customize/ai_list/index) of the **Customize** dialog box. {{pro}}
- Added the **Show suggestions only with Ctrl+Space**, **Show suggestions only when confident**, and **Set the temperature** checkboxes, **Don't show suggestions after any of the following** checkbox and text box, the **Delay time** slider, and **Additional instructions** text box to the [**AI Assist** page](../dlg/properties/ai_assist/index) of configuration properties. {{pro}}
- Added a **Show language/icon options** check box to the [**Customize Start Window** dialog box](../dlg/customize_start/index). {{profree}}
- Added the **Show additional information on status bar** check box to the [**AI Options** page](../dlg/customize/ai/index) of the **Customize** dialog box. {{pro}}

## New Macro Features {{pro}}

- The **Run macros asynchronously by default** option on the **Options** page of the **Customize Macros** dialog box is now turned off by default.
- The syntax checker is now disabled while a macro is running.
