# LineNumbersZeroBase Property (GeneralProp Object)

Corresponds to the **Show zero-based line/column numbers checkbox** check box in the
[**General** page](../../dlg/properties/general/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.LineNumbersZeroBase;
object.LineNumbersZeroBase = b;
```

### \[VBScript\]

```
b = object.LineNumbersZeroBase
object.LineNumbersZeroBase = b
```

## Version

Supported on EmEditor Professional Version 26.1 or later.
