# Save All with Encoding dialog box

This dialog box appears when the [**Save All with Encoding** command](../../cmd/file/file_save_all_as) is selected. You can save all opened documents with the specified destination folder, encoding, and newline characters, except for untitled documents.

## Destination folder text box

Specifies the destination folder.

## Encoding drop-down list box

Specifies the encoding to be used to save the files.

## Newline character drop box

Specifies the newline characters to be used to save the files.
