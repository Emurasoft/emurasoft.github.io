# 「捷徑」頁面

**捷徑** 頁面讓您能自訂與捷徑相關的設定。

## 「新增一個捷徑到「傳送到」功能表」核取方塊

在 EmEditor 中，添加一個捷徑到檔案總管中的 "發送到" 功能表中。

## 「在任務長條圖中顯示系統匣圖示」核取方塊

在 Windows 任務長條圖的通知區域顯示一個 EmEditor 的系統匣圖示。

## 「Windows 啟動時在顯示系統匣圖示」核取方塊

顯示一個系統匣圖示當 Windows 下次啟動時。

## 「EmEditor 啟動時在顯示系統匣圖示」核取方塊

顯示一個系統匣圖示當 EmEditor 下次啟動時。

## 「EmEditor 的高速啟動」核取方塊

高速啟動 EmEditor 當 EmEditor 系統匣圖示顯示時。如果勾選了該核取方塊，一個隱藏的 EmEidtor 視窗在會在 EmEditor 系統匣圖示顯示的同時運行。當您啟動 EmEditor 時，這個隱藏的 EmEidtor 視窗會變為可見的狀態，迅速打開 EmEditor。

## 「將 EmEditor 添加至網頁瀏覽器的 HTML 編輯器清單」核取方塊

將 EmEditor 添加至網頁瀏覽器使用的 HTML 編輯器清單中。

## 「添加一個捷徑至檔案總管的內容功能表中」核取方塊

在 EmEditor 中，添加一個捷徑到檔案總管的內容功能表上。

## 「添加 EmEditor 圖示到內容功能表中」核取方塊

添加 EmEditor 圖示到內容功能表上。僅當勾選了 **添加一個捷徑到檔案總管的內容功能表** 核取方塊時，才能啟用此選項。

## 「在檔案總管中添加捷徑到 Windows 11 內容功能表」核取方塊

此選項會添加捷徑到 Windows 11 檔案總管的內容功能表中。要使用此功能，EmEditor 系統匣圖示必須在 Windows 工作列的通知區中可見。僅當滿足以下所有條件時才能啟用此核取方塊：

- 操作系統是**Windows 11**。
- EmEditor 的桌面安裝程式版本正在運行。
- 勾選 **在工作列中顯示系統匣圖示** 核取方塊。
- 勾選 **Windows 啟動時顯示系統匣圖示** 核取方塊 。
- 勾選 **添加一個捷徑至檔案總管的內容功能表中** 核取方塊。

## 「自訂系統匣圖示」按鈕

顯示 [**自訂系統匣圖示** 對話方塊](../../tray/index)，讓您能自訂系統系統匣圖示設定。
