# 「自訂系統匣圖示」對話方塊

這個對話方塊會顯示當 [**自訂系統匣圖示** 命令](../../cmd/tools/customize_tray) 被選取時。

## 「單擊游標左鍵」下拉清單方塊

指定一個動作當在系統匣圖示上點擊游標左鍵。

## 「單擊游標右鍵」下拉清單方塊

指定一個動作當在系統匣圖示上點擊游標左鍵。

## 「單擊游標中鍵」下拉清單方塊

指定一個動作當在系統匣圖示上點擊游標左鍵。

## 「雙擊游標左鍵」下拉清單方塊

指定一個動作當在系統匣圖示上雙擊游標左鍵。

## 「模擬游標左鍵的快速鍵」文字方塊

為游標左鍵指定快速鍵。例如，如果您輸入 N，ALT + CTRL + N 會被指定為快速鍵。

## 「使用 EmEditor 抓取文字的快速鍵」文字方塊

指定打開一個有焦點編輯方塊內容的 EmEditor 視窗，并且原來編輯方塊中的內容會被 EmEditor 中的內容所取代當關閉 EmEditor 視窗時。例如，如果您輸入 X，ALT + CTRL + X 會被指定為快速鍵。

## 「重設」按鈕

重設為預設設定。
