# Version 26.0

2026 年 2 月 4 日  

## 一般新增功能

- 現在您可以將資料夾與檔案都加入**我的最愛**。{{profree}}
- Markdown 的 **粗體**/*斜體*/`程式碼` 命令現在即使未選取文字也能運作。{{pro}}
- **復原/重做歷史記錄**對話方塊已改為自訂列。{{profree}}  
- **插入特殊字元**對話方塊已改為自訂列，並改進包含更大的彩色字型，以及可依 Unicode 名稱與區塊進行篩選。{{profree}} 
- **字元碼值**對話方塊已改進，支援更大字型與色彩。{{profree}} 
- 改善了彆扭的英文、德文與日文 UI 文字。{{profree}} 
- 在 v25.4 中已停止提供俄文 UI。{{profree}}
- 提升了多項操作的速度，包括**開啟不含換行的大型檔案（單行檔）**。{{profree}}

### 測試結果：

|  | v25.4 | v26.0 | 與 v25.4 相比 |
| --- | --- | --- | --- |
| 開啟無換行的大型檔案 [\*a\] | 1.426 秒 | 0.619 秒 | 快 2.30 倍 |
| 尋找（全選）[\*b\] | 43.3 秒 | 0.065 秒 | 快 666 倍 |
| 全部取代 [\*c\] | 44.1 秒 | 0.359 秒 | 快 123 倍 |

- \*a：開啟檔案。
- \*b：全選所有「abc」。
- \*c：將所有「abc」取代為「###」。
- \*All：Wikipedia 傾印檔（UTF-8，1.98 GB，無換行）、所有預設選項、Windows 11、Ryzen 9 7950X、96 GB RAM。

## 新增選項

- 在**自訂**對話方塊的[**我的最愛** 頁面](../dlg/customize/favorites/index)新增**新增資料夾**按鈕。{{profree}}
- 在**自訂**對話方塊的 [**AI 選項** 頁面](../dlg/customize/ai/index)的**首選模型**下拉清單新增 `gpt-5.2` 與 `gpt-5.2.codex`（此功能需要 **與 AI 對話** 外掛程式）。{{pro}}
