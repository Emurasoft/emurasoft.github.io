# Copy 方法 (Selection H)

複製選定內容到剪貼簿上。

#### \[JavaScript\]

document.selection. **Copy**( _nFlags_ );

#### \[VBScript\]

document.selection. **Copy** _nFlags_

## 參數

_nFlags_

指定一個下列值的組合:

|     |     |
| --- | --- |
| eeCopyUnicode | 複製為 Unicode (預設) 。 |
| eeCopyQuotes | 複製為引用文字。 |

## 版本

支持 EmEditor 4.00 或之後的版本。
