# FindRepeat 方法 (Selection 對象)

對同一個字串重複上一次搜尋。

## 

### \[JavaScript\]

```
document.selection.FindRepeat( nFlags );
```

### \[VBScript\]

```
document.selection.FindRepeat nFlags
```

## 參數

_nFlags_

指定一個下列值的組合:

|     |     |
| --- | --- |
| eeFindRepeatNext | 從游標處往下搜尋字串。 |
| eeFindRepeatPrevious | 從游標處往上搜尋字串。 |
| eeFindRepeatWord | 搜尋游標處的單字如果選定內容為空的話。 |

## 版本

支持 EmEditor 4.00 或之後的版本。
