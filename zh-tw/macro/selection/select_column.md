# SelectColumn 方法 (Selection 對象)

在 CSV 模式中選擇指定的欄。

## 

### \[JavaScript\]

```
document.selection.SelectColumn( iColumn, [ [ iColumn2 ], nFlags ] );
```

### \[VBScript\]

```
document.selection.SelectColumn iColumn, [ [ iColumn2 ], nFlags ]
```

## 參數

_nFlags_

指定 0 或以下值：

|     |     |
| --- | --- |
| eeNoHeadings | 選取內容排除標題。 |

## 返回值

此方法不返回值。

## 版本

支持 EmEditor Professional 21.0 或之後的版本。
