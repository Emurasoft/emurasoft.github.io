# LineOpen 方法 (Selection H)

在兩行之間插入一個空行。

#### \[JavaScript\]

document.selection. **LineOpen**( \[ _bAbove_ \] );

#### \[VBScript\]

document.selection. **LineOpen** \[ _bAbove_ \]

## 參數

_bAbove_

可選項。如果 true 被指定，在目前的行的開始插入一個新行。如果 false 被指定，在目前的行的末尾插入一個新行。預設值是 false。

## 版本

支持 EmEditor 4.00 或之後的版本。
