# LineNumbersZeroBase 屬性 (GeneralProp 對象)

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **顯示基底為 0 的行/欄號** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.LineNumbersZeroBase;
object.LineNumbersZeroBase = b;
```

### \[VBScript\]

```
b = object.LineNumbersZeroBase
object.LineNumbersZeroBase = b
```

## 版本

支持 EmEditor Professional 26.1 或更高的版本。
