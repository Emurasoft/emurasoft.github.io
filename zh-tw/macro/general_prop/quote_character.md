# QuoteCharacter 屬性 (GeneralProp 對象)

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **引用字元** 文字方塊相對應。

## 

### \[JavaScript\]

```
str = object.QuoteCharacter;
object.QuoteCharacter = str;
```

### \[VBScript\]

```
str = object.QuoteCharacter
object.QuoteCharacter = str
```

## 版本

支持 EmEditor 7.00 或之後的版本。
