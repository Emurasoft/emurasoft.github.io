# write 方法 (Interface 對象)

把一個字串傳遞到一個支持的外掛程式中。

## 

### \[JavaScript\]

```
Interface.write( str );
```

### \[VBScript\]

```
Interface.write str
```

## 參數

_str_

指定要傳遞的字串。

## 示例

### \[JavaScript\]

```
Interface.write( "Hello" );
```

### \[VBScript\]

```
Interface.write "Hello"
```

## 版本

支持 EmEditor 9.00 或之後的版本。
