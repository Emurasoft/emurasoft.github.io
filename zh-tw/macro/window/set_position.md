# SetPosition 方法 (Window H)

設置視窗的大小和位置。

#### \[JavaScript\]

wnd. **SetPosition**( _x_, _y_, _width_, _height_ );

#### \[VBScript\]

wnd. **SetPosition** _x_, _y_, _width_, _height_

## 參數

_x_

以像素為單位指定視窗的水平位置。

_y_

以像素為單位指定視窗的垂直位置。

_width_

以像素為單位指定視窗的寬度。

_height_

以像素為單位指定視窗的高度。

## 版本

支持 EmEditor 7.00 或之後的版本。
