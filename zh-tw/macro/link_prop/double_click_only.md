# DoubleClickOnly 屬性 (LinkProp 對象)

與組態屬性中 [連結 頁面](../../dlg/properties/link/index) 上的僅允許雙擊 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.DoubleClickOnly;
object.DoubleClickOnly = b;
```

### \[VBScript\]

```
b = object.DoubleClickOnly
object.DoubleClickOnly = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
