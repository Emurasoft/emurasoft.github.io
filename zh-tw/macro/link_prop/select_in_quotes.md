# SelectInQuotes 屬性

與組態屬性中 [**連結** 頁面](../../dlg/properties/link/index) 上的 **點擊引號內的文字選擇整個字串** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **SelectInQuotes**;

object. **SelectInQuotes** = _b_;

#### \[VBScript\]

_b_ =
object. **SelectInQuotes**

object. **SelectInQuotes** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
