# URLCharactersNotEnd 屬性 (LinkProp 對象)

與組態屬性中 [連結 頁面](../../dlg/properties/link/index) 上的行末不允許的字元 核取方塊相對應。

## 

### \[JavaScript\]

```
s = object.URLCharactersNotEnd;
object.URLCharactersNotEnd = s;
```

### \[VBScript\]

```
s = object.URLCharactersNotEnd
object.URLCharactersNotEnd = s
```

## 版本

支持 EmEditor 7.00 或之後的版本。
