# HighlightCharRef 屬性 (HighlightProp 對象)

與組態屬性中 [**亮顯 (2)** 頁面](../../dlg/properties/highlight2/index) 上的 **亮顯 HTML/XML 中的字元引用及通用字元名稱** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.HighlightCharRef;
object.HighlightCharRef = b;
```

### \[VBScript\]

```
b = object.HighlightCharRef
object.HighlightCharRef = b
```

## 版本

支持 EmEditor 11.00 或之後的版本。
