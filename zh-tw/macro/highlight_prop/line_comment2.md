# LineComment2 屬性 (HighlightProp 對象)

與組態屬性中 [**亮顯 (2)** 頁面](../../dlg/properties/highlight2/index) 上的 **「注釋」群組方塊 \- 「行注釋」** 文字方塊相對應。

## 

### \[JavaScript\]

```
str = object.LineComment2;
object.LineComment2 = str;
```

### \[VBScript\]

```
str = object.LineComment2
object.LineComment2 = str
```

## 版本

支持 EmEditor 7.00 或之後的版本。
