# SpecialSyntax 屬性 (HighlightProp 對象)

與組態屬性中 [**亮顯 (2)** 頁面](../../dlg/properties/highlight2/index) 上的 **特殊語法** 下拉清單方塊相對應。

## 

### \[JavaScript\]

```
n = object.SpecialSyntax;
object.SpecialSyntax = n;
```

### \[VBScript\]

```
n = object.SpecialSyntax
object.SpecialSyntax = n
```

## 版本

支持 EmEditor 7.00 或之後的版本。
