# InsertSpaces 屬性

與 [**Tab/縮排** 對話方塊](../../dlg/properties/general/indent/index) 上的 **將 Tab轉換為空格** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **InsertSpaces**;

object. **InsertSpaces** = _b_;

#### \[VBScript\]

_b_ =
object. **InsertSpaces**

object. **InsertSpaces** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
