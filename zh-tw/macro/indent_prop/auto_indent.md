# AutoIndent 屬性 (IndentProp 對象)

與 [**Tab/縮排** 對話方塊](../../dlg/properties/general/indent/index) 上的 **啟用自動縮排** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.AutoIndent;
object.AutoIndent = b;
```

### \[VBScript\]

```
b = object.AutoIndent
object.AutoIndent = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
