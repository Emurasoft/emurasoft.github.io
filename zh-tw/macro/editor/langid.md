# LangID 屬性 (Editor 對象)

檢索目前選取的語言 ID，即 Microsoft 指定的 [Locale IDs](https://docs.microsoft.com/en-us/openspecs/windows_protocols/ms-lcid/a9eac961-e77d-41a6-90a5-ce1a8b0cdb9c)。

## 

### \[JavaScript\]

```
nID = editor.LangID;
```

### \[VBScript\]

```
nID = editor.LangID
```

## 版本

支持 EmEditor Professional 15.5 或之後的版本。
