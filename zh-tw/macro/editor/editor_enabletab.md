# EnableTab 屬性 (Editor 對象)

設置或檢索是否標簽頁被啟用。所有的文檔可能無法立即使用在標簽頁被啟用或禁用后。因此，在標簽頁被啟用或禁用后，可能有必要先退出巨集再運行該巨集。

## 

### \[JavaScript\]

```
bEnableTab = editor.EnableTab;
editor.EnableTab = bEnableTab;
```

### \[VBScript\]

```
bEnableTab = editor.EnableTab
editor.EnableTab = bEnableTab
```

## 版本

支持 EmEditor 5.00 或之後的版本。
