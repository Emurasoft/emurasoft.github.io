# AddCharacterRange 方法 (CharacterRanges 集合)

添加字元範圍。

## 

### \[JavaScript\]

```
list.AddCharacterRange( nMin, nMax, sProperties, sTreatAs );
```

### \[VBScript\]

```
list.AddCharacterRange nMin, nMax, sProperties, sTreatAs
```

## 參數

_nMin_

指定範圍的最小字元。

_nMax_

指定範圍的最大字元。

_sProperties_

指定表示 Unicode 指令碼和/或 Unicode 一般類別屬性的字串。

_sTreatAs_

指定要取代的字串。如果指定了空字串，則指定的範圍將被忽略。

## 版本

支持 EmEditor Professional Version 22.0 或之後的版本。
