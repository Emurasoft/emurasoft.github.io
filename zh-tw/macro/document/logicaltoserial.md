# LogicalToSerial 方法 (Document H)

將指定位置的邏輯坐標轉換為基于單個的串行位置。

## 

### \[JavaScript\]

```
nSerialPos = document.LogicalToSerial( x, y );
```

### \[VBScript\]

```
nSerialPos = document.LogicalToSerial( x, y )
```

## 參數

_x_

指定基于單個的水平（字元）位置。

_y_

指定基于單個的垂直（行）位置。

## 範例

### \[JavaScript\]

```
n = document.LogicalToSerial( 5, 1 );
```

### \[VBScript\]

```
n = document.LogicalToSerial( 5, 1 )
```

## 版本

支持 EmEditor Professional 17.0 或之後的版本。
