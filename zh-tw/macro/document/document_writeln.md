# writeln 方法 (Document 對象)

在當前光標位置處插入或改寫一個字符串并換行。

## 

### \[JavaScript\]

```
document.writeln( str );
```

### \[VBScript\]

```
document.writeln str
```

## 參數

_str_

指定要插入或改寫為的字符串。

## 示例

### \[JavaScript\]

```
document.writeln( "Hello" );
```

### \[VBScript\]

```
document.writeln "Hello"
```

## 版本

支持 EmEditor 4.00 或之後的版本。
