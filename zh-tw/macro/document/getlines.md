# GetLines 方法

檢索文檔中的行數。

#### \[JavaScript\]

_nLines_ = document. **GetLines**( \[ _nFlags_ \] );

#### \[VBScript\]

_nLines_ = document. **GetLines**( \[ _nFlags_ \] )

## 參數

_nFlags_

可選項。您能指定下列值。如果您沒有指定任何值，EmEditor 會預設指定邏輯坐標。

|     |     |
| --- | --- |
| eeGetLineView | 在視圖中指定坐標。 |

## 版本

支持 EmEditor 7.00 或之後的版本。
