# ForegroundWindow 屬性 (Shell 對象)

檢索當前置前視窗。

## 

### \[JavaScript\]

```
wnd = shell.ForegroundWindow;
```

### \[VBScript\]

```
wnd = shell.ForegroundWindow
```

## 示例

### \[JavaScript\]

```
wnd = shell.FouregroundWindow;
str = wnd.Caption;
alert( str );
```

### \[VBScript\]

```
wnd = shell.FouregroundWindow
str = wnd.Caption
alert str
```

## 版本

支持 EmEditor 7.00 或之後的版本。
