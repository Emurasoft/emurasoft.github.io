# ShowFullPath 屬性 (FileProp H)

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **顯示包含完整路徑的檔案名** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **ShowFullPath**;

object. **ShowFullPath** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowFullPath**

object. **ShowFullPath** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
