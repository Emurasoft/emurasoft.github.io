# MonitorInterval 屬性 (FileProp 對象)

與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的監控時間間隔 文字方塊相對應。

## 

### \[JavaScript\]

```
n = object.MonitorInterval;
object.MonitorInterval = n;
```

### \[VBScript\]

```
n = object.MonitorInterval
object.MonitorInterval = n
```

## 版本

支持 EmEditor 7.00 或之後的版本。
