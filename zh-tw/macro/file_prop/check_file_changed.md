# CheckFileChanged 屬性

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **被其他程式更改時** 下拉清單方塊相對應。

#### \[JavaScript\]

_n_ = object. **CheckFileChanged**;

object. **CheckFileChanged** = _n_;

#### \[VBScript\]

_n_ = object. **CheckFileChanged**

object. **CheckFileChanged** = _n_

## 版本

支持 EmEditor 7.00 或之後的版本。
