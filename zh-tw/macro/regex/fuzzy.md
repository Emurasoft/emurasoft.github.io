# Fuzzy 屬性 (Regex 對象)

設定或檢索模糊比對標志。此特殊標志使用模糊比對，並停用規則運算式。你不能將模糊比對與規則運算式結合使用。如果設定了此屬性，會忽略[Engine](engine) 和[SeparateCrLf](separate_cr_lf) 屬性。

## 

### \[JavaScript\]

```
b = reg.Fuzzy;
reg.Fuzzy = b;
```

### \[VBScript\]

```
b = reg.Fuzzy;
reg.Fuzzy = b;
```

## 版本

支持 EmEditor Professional Version 22.0 或之後的版本。
