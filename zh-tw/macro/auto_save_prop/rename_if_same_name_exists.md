# RenameIfSameNameExists 屬性 (AutoSaveProp 對象)

與組態屬性中 [**自動儲存** 頁面](../../dlg/properties/autosave/index) 上的 **重新命名檔案，檔案名已存在** 核取方塊對應。

## 

### \[JavaScript\]

```
b = object.RenameIfSameNameExists;
object.RenameIfSameNameExists = b;
```

### \[VBScript\]

```
b = object.RenameIfSameNameExists
object.RenameIfSameNameExists = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
