# ShowIndentGuides 屬性 (MarkProp 對象)

與組態屬性中 [標記 頁面](../../dlg/properties/marks/index) 上的顯示縮排線 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.ShowIndentGuides;
object.ShowIndentGuides
= b;
```

### \[VBScript\]

```
b = object.ShowIndentGuides
object.ShowIndentGuides
= b
```

## 版本

支持 EmEditor 8.00 或之後的版本。
