# ShowControlCharacters 屬性

與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示控制字元** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **ShowControlCharacters**;

object. **ShowControlCharacters** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowControlCharacters**

object. **ShowControlCharacters** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
