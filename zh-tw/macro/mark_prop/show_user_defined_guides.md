# ShowUserDefinedGuides 屬性 (MarkProp 對象)

與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示使用者自訂參考線** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.ShowUserDefinedGuides;
object.ShowUserDefinedGuides = b;
```

### \[VBScript\]

```
b = object.ShowUserDefinedGuides
object.ShowUserDefinedGuides = b
```

## 版本

支持 EmEditor 14.2.0 或之後的版本。
