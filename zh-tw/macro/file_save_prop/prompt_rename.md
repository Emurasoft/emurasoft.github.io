# PromptRename 屬性

與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **更名時在刪除舊檔案前提示** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **PromptRename**;

object. **PromptRename** = _b_;

#### \[VBScript\]

_b_ =
object. **PromptRename**

object. **PromptRename** = _b_

## 版本

支持 EmEditor 13 或之後的版本。
