# NumericEntity 屬性 (FileSaveProp 對象)

與 [儲存詳細信息 對話方塊](../../dlg/properties/file/save_details/index) 中的把 Unicode 儲存為 HTML/XML 字元引用 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.NumericEntity;
object.NumericEntity = b;
```

### \[VBScript\]

```
b = object.NumericEntity
object.NumericEntity = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
