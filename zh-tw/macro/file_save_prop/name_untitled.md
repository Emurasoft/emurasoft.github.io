# NameUntitled 屬性

與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **自動為無標題文檔命名** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **NameUntitled**;

object. **NameUntitled** = _b_;

#### \[VBScript\]

_b_ =
object. **NameUntitled**

object. **NameUntitled** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
