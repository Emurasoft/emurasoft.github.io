# ShowOnlyActive 屬性 (ScrollProp 對象)

與組態屬性中 [捲動 頁面](../../dlg/properties/scroll/index) 上的只在活動窗格中顯示捲軸 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.ShowOnlyActive;
object.ShowOnlyActive = b;
```

### \[VBScript\]

```
b = object.ShowOnlyActive
object.ShowOnlyActive = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
