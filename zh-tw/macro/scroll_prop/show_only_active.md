# ShowOnlyActive 屬性

與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **只在活動窗格中顯示捲軸** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **ShowOnlyActive**;

object. **ShowOnlyActive** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowOnlyActive**

object. **ShowOnlyActive** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
