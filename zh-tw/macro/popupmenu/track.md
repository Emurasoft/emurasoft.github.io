# Track 方法 (PopupMenu H)

顯示快顯功能表，檢索所選功能表項的識別項。

#### \[JavaScript\]

id = popupmenu. **Track**( _flags_ );

#### \[VBScript\]

id = popupmenu. **Track**( _flags_ )

## 參數

_flags_

指定菜單要顯示的位置。能指定下列值。如果不指定，彈出菜單會在當前光標位置處顯示。

|     |     |
| --- | --- |
| eePosMouse | 在當前鼠標指針位置處顯示彈出菜單。 |

## 版本

支持 EmEditor 5.00 或之後的版本。
