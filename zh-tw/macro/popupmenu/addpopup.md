# AddPopup 方法 (PopupMenu 對象)

添加一個新的子功能表到功能表末尾。

## 

### \[JavaScript\]

```
popupmenu.AddPopup( strText, submenu);
```

### \[VBScript\]

```
popupmenu.AddPopup strText, submenu
```

## 參數

_strText_

指定要顯示的文字。

_submenu_

指定一個分隔的 [PopupMenu 對象](index) 來作為一個新的子功能表顯示。

## 版本

支持 EmEditor 5.00 或之後的版本。
