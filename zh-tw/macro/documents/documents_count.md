# Count 屬性 (Documents 集合)

檢索文檔的數目。

## 

### \[JavaScript\]

```
nCount = editor.Documents.Count;
```

### \[VBScript\]

```
nCount = editor.Documents.Count
```

## 示例

### \[JavaScript\]

```
alert( "Number of Documents: " + editor.Documents.Count );
```

### \[VBScript\]

```
alert "Number of Documents: " & editor.Documents.Count
```

## 版本

支持 EmEditor 5.00 或之後的版本。
