# SaveToSameFolder 屬性 (BackupProp 對象)

與組態屬性中 [**備份** 頁面](../../dlg/properties/backup/index) 上的 **儲存備份檔案到源檔案所在資料夾** 核取方塊對應。

## 

### \[JavaScript\]

```
b = object.SaveToSameFolder;
object.SaveToSameFolder = b;
```

### \[VBScript\]

```
b = object.SaveToSameFolder
object.SaveToSameFolder = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
