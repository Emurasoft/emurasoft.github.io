# Windows Scripting Host (\)

巨集的操作基于 Windows Scripting Host (WSH) 引擎的工作環境。因此，您可以用不同的Windows Component 對象 Model (COM) 對象。例如您可以用 RegExp 對象來執行規則運算式搜尋，用 FileSystemObject 對象來操控檔案，用 WshShell 對象創建快捷鍵，操縱 Windows 注冊表，或者用 WshNetwork 對象來進行具有聯網功能的工作。另外，您可以創建并執行巨集，讓您能用一個支持自動化 (例如 Word 和 Excel) 的外部程序來複製 EmEditor 中創建的文檔，粘貼并顯示在外部程序中。
