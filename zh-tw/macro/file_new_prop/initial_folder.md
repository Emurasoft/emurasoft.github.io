# InitialFolder 屬性

與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **初始資料夾** 文字方塊相對應。

#### \[JavaScript\]

_s_ = object. **InitialFolder**;

object. **InitialFolder** = _s_;

#### \[VBScript\]

_s_ = object. **InitialFolder**

object. **InitialFolder** = _s_

## 版本

支持 EmEditor 14.1 或之後的版本。
