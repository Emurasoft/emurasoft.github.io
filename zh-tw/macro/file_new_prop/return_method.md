# ReturnMethod 屬性 (FileNewProp 對象)

與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **換行方式** 下拉清單方塊相對應。

## 

### \[JavaScript\]

```
n = object.ReturnMethod;
object.ReturnMethod = n;
```

### \[VBScript\]

```
n = object.ReturnMethod
object.ReturnMethod = n
```

## 版本

支持 EmEditor 7.00 或之後的版本。
