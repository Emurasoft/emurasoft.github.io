# ShowLineNumbers 屬性 (PrintProp 對象)

與組態屬性中 [列印 頁面](../../dlg/properties/print/index) 上的列印行號 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.ShowLineNumbers;
object.ShowLineNumbers = b;
```

### \[VBScript\]

```
b = object.ShowLineNumbers
object.ShowLineNumbers = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
