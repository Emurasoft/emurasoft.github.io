# TextMode 屬性 (DisplayItem 對象)

指定文字顏色模式。

## 

### \[JavaScript\]

```
nMode = item.TextMode;
item.TextMode = nMode;
```

### \[VBScript\]

```
nMode = item.TextMode
item.TextMode = nMode
```

## 備注

nMode 參數會是下列值之一。

|     |     |
| --- | --- |
| 0 | 自訂 |
| 1 | 預設 |
| 2 | 透明 |

## 版本

支持 EmEditor 7.00 或之後的版本。
