# 規則運算式搜尋範例

- 被雙引號包圍的字串

".\*?"
- 被 \[ \] 包圍的字串

\\\[\[^\\\[\]\*?\\\]
- 變數名稱

\[a-zA-Z\_\]\[a-zA-Z\_0-9\]\*
- IP 位址

(\[0-9\]{1,3})\\.(\[0-9\]{1,3})\\.(\[0-9\]{1,3})\\.(\[0-9\]{1,3})
- URL

(\\S+)://(\[^:/\]+)(:(\\d+))?(/\[^#\\s\]\*)(#(\\S+))?
- 含有 Tab 的行

\\t.\*$
- 平假名

\[\\x{3041}-\\x{309e}\]
- 全形片假名

\[\\x{309b}-\\x{309c}\\x{30a1}-\\x{30fe}\]
- 半形片假名

\[\\x{ff61}-\\x{ff9f}\]
- 中日韓統一表意文字

\[\\x{3400}-\\x{9fff}\\x{f900}-\\x{fa2d}\]
- 中日韓表意文字記號

\[\\x{3000}-\\x{3037}\]
- 韓文

\[\\x{1100}-\\x{11f9}\\x{3131}-\\x{318e}\\x{ac00}-\\x{d7a3}\]
- 在行首插入 //

尋找：^

取代為：//
- 移除行首的 //

尋找：^//

取代：
- 移除行尾空白

尋找：\\s+?$

取代為：
- 將 (abc) 取代為 \[abc\]

尋找：\\((.\*?)\\)

取代：\\\[\\1\\\]
- 將 <H3 ...> 取代為 <H4 ...>

尋找：<H3(.\*?)>

取代：<H4\\1>
- 將 9/13/2003 取代為 2003.9.13

尋找：(\[0-9\]{1,2})/(\[0-9\]{1,2})/(\[0-9\]{2,4})

取代：\\3\\.\\1\\.\\2
- 將 a 到 z 的字母轉為大寫

尋找：\[a-z\]

取代為：\\U\\0
- 將所有單字首字母大寫

尋找：(\[a-zA-Z\])(\[a-zA-Z\]\*)

取代：\\U\\1\\L\\2
