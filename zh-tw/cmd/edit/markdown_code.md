# 程式碼命令

## 摘要

切換程式碼樣式。

## 說明

切換程式碼樣式。

## 運行方法

- 預設功能表: **編輯** \> **Markdown** \> **程式碼**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown** \> **程式碼**
- 工具列: ![](../../images/markdown_code..png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_CODE (23278)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23278);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23278
```
