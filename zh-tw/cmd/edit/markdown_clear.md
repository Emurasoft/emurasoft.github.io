# 清除所有格式命令

## 摘要

清除所選文字的粗體和斜體格式。

## 說明

清除所選文字的粗體和斜體格式。

## 運行方法

- 預設功能表: **編輯** \> **Markdown** \> **清除所有格式**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown** \> **清除所有格式**
- 工具列: ![](../../images/markdown_clear.gif)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_CLEAR (23276)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23276);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23276
```
