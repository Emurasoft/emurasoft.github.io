# 標準模式命令

## 摘要

選擇標準模式。

## 說明

清除 CSV (逗號分隔)，TSV (Tab分隔)，和 DSV (使用者定義的分隔符分隔) 模式。

## 運行方法

- 預設功能表:編輯 \>CSV \>標準模式
- [全部命令](../tools/all_commands):編輯 \>CSV \>標準模式
- 工具列: ![](../../images/default_mode.gif)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_MODE_NORMAL (4472)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4472);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4472
```
