# 跳轉命令

## 摘要

跳轉到指定行。

## 說明

這個命令會顯示 [「跳轉」 對話方塊](../../dlg/jump/index)。您可以在這個對話方塊中指定一個行號，文檔會跳轉到指定的行上。

## 運行方法

- 預設功能表:編輯 \>跳轉
- [全部命令](../tools/all_commands):編輯 \>進階
\>跳轉
- 工具列: ![](../../images/jump.gif)
- 狀態列: 無
- 預設捷徑: CTRL+G

## 外掛程式命令ID

```
EEID_JUMP (4139)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4139);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4139
```
