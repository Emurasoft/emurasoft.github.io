# 超連結命令

## 摘要

插入超連結。

## 說明

插入超連結。

## 運行方法

- 預設功能表: **編輯** \> **Markdown/HTML** \> **超連結**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown/HTML** \> **超連結**
- 工具列: ![](../../images/hyperlink.png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_HYPERLINK (23271)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23271);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23271
```
