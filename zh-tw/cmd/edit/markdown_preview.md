# Markdown 預覽命令

## 摘要

顯示目前的文檔的 Markdown 預覽。

## 說明

顯示目前的文檔的 Markdown 預覽。

## 運行方法

- 預設功能表: **編輯** \> **Markdown/HTML** \> **預覽**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown/HTML** \> **預覽**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_PREVIEW (23275)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23275);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23275
```
