# 全部復原命令

## 摘要

復原所有之前的操作。

## 說明

復原所有之前的操作。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **復原** \> **全部復原**
- 工具列: ![](../../images/editundo.gif) (向下箭頭)
- 狀態列: 無
- 預設捷徑: 無

```
EEID_UNDO_ALL (3980)```

## 巨集

editor.ExecuteCommandByID(3980);
