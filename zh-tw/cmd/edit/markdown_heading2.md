# 二級標題命令

## 摘要

將目前的行設定為二級標題。

## 說明

將目前的行設定為二級標題。

## 運行方法

- 預設功能表: **編輯** \> **Markdown/HTML** \> **二級標題**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown/HTML** \> **二級標題**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_HEADING2 (23258)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23258);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23258
```
