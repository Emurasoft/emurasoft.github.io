# 區塊引用命令

## 摘要

將目前的行設定為區塊引用。

## 說明

將目前的行設定為區塊引用。

## 運行方法

- 預設功能表: **編輯** \> **Markdown/HTML** \> **區塊引用**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown/HTML** \> **區塊引用**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_BLOCKQUOTE (23263)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23263);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23263
```
