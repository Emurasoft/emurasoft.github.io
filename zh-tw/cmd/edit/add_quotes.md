# 添加引號到所有儲存格上命令

### 摘要

> 添加雙引號到 CSV 文檔的所有儲存格上。

### 說明

> 添加雙引號到 CSV 文檔的所有儲存格上。

### 運行方法

- 預設功能表: **CSV** \> **進階** \> **添加引號到所有儲存格上**
- [全部命令](../tools/all_commands): **CSV** \> **進階** \> **添加引號到所有儲存格上**
- 工具列: ![](../../images/convert_to.gif) \+ 添加引號到所有儲存格上
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_ADD\_QUOTES (3966)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3966);

#### \[VBScript\]

> editor.ExecuteCommandByID 3966
