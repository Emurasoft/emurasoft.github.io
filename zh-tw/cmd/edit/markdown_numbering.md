# 編號命令

## 摘要

插入編號清單。

## 說明

插入編號清單。

## 運行方法

- 預設功能表: **編輯** \> **Markdown** \> **編號**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown** \> **編號**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_NUMBERING (23265)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23265);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23265
```
