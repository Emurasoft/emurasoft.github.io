# 項目符號命令

## 摘要

插入項目符號清單。

## 說明

插入項目符號清單。

## 運行方法

- 預設功能表: **編輯** \> **Markdown** \> **項目符號**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown** \> **項目符號**
- 工具列: ![](../../images/bullets..png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_BULLETS (23266)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23266);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23266
```
