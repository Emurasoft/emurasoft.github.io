# 按字母降序命令

## 摘要

在目前的欄中以字母降序排序文字。

## 說明

在目前的欄中以字母降序排序文字。如果文字不包含字母字元 (A-Z和a-z) ，這個命令會嘗試按數字值排序。

## 運行方法

- 預設功能表: **排序** \> **Z 到 A 排序**
- [全部命令](../tools/all_commands): **排序** \> **Z 到 A 排序**
- 工具列: ![](../../images/sortingz-a.gif)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_TEXT_D (4478)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4478);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4478
```
