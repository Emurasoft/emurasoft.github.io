# 分行符號命令

## 摘要

插入分行符號。

## 說明

插入分行符號。

## 運行方法

- 預設功能表: **編輯** \> **Markdown/HTML** \> **分行符號**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown/HTML** \> **分行符號**
- 工具列: ![](../../images/markdown_line_break.png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_LINE_BREAK (23268)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23268);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23268
```
