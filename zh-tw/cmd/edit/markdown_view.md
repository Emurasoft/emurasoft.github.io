# Markdown 設計檢視命令

## 摘要

顯示 Markdown 設計檢視。

## 說明

顯示 Markdown 設計檢視。

## 運行方法

- 預設功能表: **編輯** \> **Markdown/HTML** \> **設計檢視**
- [全部命令](../tools/all_commands): **編輯** \> **Markdown/HTML** \> **設計檢視**
- 工具列: ![](../../images/markdown_view.png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_VIEW (23255)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23255);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23255
```
