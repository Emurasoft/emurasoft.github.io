# 同步捲動命令

## 摘要

不指定參數直接同步捲動兩個最近訪問的文檔。

## 說明

不指定參數直接同步捲動兩個最近訪問的文檔。

## 運行方法

- 預設功能表: **比較** \> **同步捲動**
- [全部命令](../tools/all_commands): **比較** \> **同步捲動**
- 工具列:  ![](../../images/sync24x16.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SYNC_SCROLL_DIRECT (4495)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4495);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4495
```
