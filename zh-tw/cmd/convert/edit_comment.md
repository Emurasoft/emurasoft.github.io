# 註解命令

## 摘要

註解選定區域或目前的行。

## 說明

通過在選定區域的開頭和末尾插入註解標記來對選取區域進行注解。如果文檔中沒有任何選取，這個命令會對目前的行進行注解。這個命令不適用於僅文字的組態。

## 運行方法

- 預設功能表: **轉換** \> **註解**
- [全部命令](../tools/all_commands): **轉換** \> **註解**
- 工具列: ![](../../images/editcomment.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_EDIT_COMMENT (4371)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4371);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4371
```
