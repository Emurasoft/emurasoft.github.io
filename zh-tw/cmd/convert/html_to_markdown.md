# HTML 轉 Markdown 命令

## 摘要

將選取的文字從 HTML 轉換為 Markdown。

## 說明

將選取的文字從 HTML 轉換為 Markdown。

## 運行方法

- 預設功能表: **轉換** \> **編碼/解碼所選內容** \> **HTML 轉 Markdown**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼所選內容** \> **HTML 轉 Markdown**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_HTML_TO_MARKDOWN (23281)
```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(23281);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23281
```
