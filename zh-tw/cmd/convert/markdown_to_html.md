# Markdown 轉 HTML 命令

## 摘要

將選取的文字從 Markdown 轉換為 HTML。

## 說明

將選取的文字從 Markdown 轉換為 HTML。

## 運行方法

- 預設功能表: **轉換** \> **編碼/解碼所選內容** \> **Markdown 轉 HTML**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼所選內容** \> **Markdown 轉 HTML**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_TO_HTML (23280)
```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(23280);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23280
```
