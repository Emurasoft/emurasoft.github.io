# HTML 轉純文字命令

## 摘要

將選取的文字從 HTML 轉換為純文字。

## 說明

將選取的文字從 HTML 轉換為純文字。

## 運行方法

- 預設功能表: **轉換** \> **編碼/解碼所選內容** \> **HTML 轉純文字**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼所選內容** \> **HTML 轉純文字**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_MARKDOWN_TO_TEXT (23283)
```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(23283);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23283
```
