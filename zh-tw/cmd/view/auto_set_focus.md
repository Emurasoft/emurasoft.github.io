# 自動設置焦點（儲存格工具列）命令

## 摘要

自動設置焦點到儲存格工具列上。

## 說明

自動設置焦點到儲存格工具列上。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** >
**工具列** \> **儲存格工具列選項** \> **自動設置焦點**
- 工具列: ![](../../images/commonsettings.png) (儲存格工具列) \+ 自動設置焦點
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CELL_BAR_AUTO_FOCUS (3970)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3970);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3970
```
