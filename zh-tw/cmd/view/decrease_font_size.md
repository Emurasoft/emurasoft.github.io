# 減小字型命令

## 摘要

減小顯示的字型大小。

## 說明

減小顯示的字型大小。字型名稱和實際大小可以在 [**自訂字型** 對話方塊](../../dlg/properties/font/index) 中指定。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **字型** >
**減小顯示的字型大小**
- 工具列: ![](../../images/decreasefontsize.gif)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_DECREASE_FONT_SIZE (4374)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4374);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4374
```
