# 尺規命令

## 摘要

顯示或隱藏尺規。

## 說明

顯示或隱藏尺規。在預設設定下，EmEditor 不顯示尺規。

## 運行方法

- 預設功能表: **檢視** \> **尺規**
- [全部命令](../tools/all_commands): **檢視** \> **尺規**
- 工具列:
![](../../images/ruler24x16..png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_VIEW_LINE_RULER (4531)
```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4531);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4531
```
