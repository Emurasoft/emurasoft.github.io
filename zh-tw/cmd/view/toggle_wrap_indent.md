# 換行縮排命令

## 摘要

啟用或停用換行縮排。

## 說明

啟用或停用換行縮排。當啟用這個命令時，換行時將執行縮排。

## 運行方法

- 預設功能表: **檢視** \> **Tab/縮排** \> **換行縮排**
- [全部命令](../tools/all_commands): **檢視** \> **Tab/縮排** \> **換行縮排**
- 工具列:
![](../../images/wrap_indent24x16.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_TOGGLE_WRAP_INDENT (4542)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4542);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4542
```
