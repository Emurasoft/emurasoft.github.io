# 切換大綱顯示條和導引命令

## 摘要

顯示或隱藏大綱顯示條和大綱導引。

## 說明

顯示或隱藏大綱顯示條和大綱導引。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **大綱** \> **大綱顯示條和導引**
- 工具列: **![Outline](../../images/plugin_outline.png)**
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_OUTLINE_BAR_GUIDE (3874)
```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3874);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3874
```
