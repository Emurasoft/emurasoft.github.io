# 程式碼片段工具列命令

## 摘要

顯示或隱藏程式碼片段工具列。

## 說明

顯示或隱藏程式碼片段工具列。

## 運行方法

- 預設功能表：**檢視** > **工具列** \> **程式碼片段**
- [所有命令](../tools/all_commands)：**檢視** > **工具列** \> **程式碼片段**
- 工具列：無
- 狀態列：無
- 預設快速鍵：無

## 外掛程式命令ID

```
EEID_SHOW_SNIPPETS_BAR (23287)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23287);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23287
```
