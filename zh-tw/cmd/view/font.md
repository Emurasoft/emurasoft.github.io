# 自訂字型命令

## 摘要

更改字型設定。

## 說明

這個命令顯示 [自訂字型 對話方塊](../../dlg/properties/font/index)，讓您可以設定顯示的字型和列印字型。

## 運行方法

- 預設功能表:檢視 \>字型
- [全部命令](../tools/all_commands):檢視 >
字型 \>字型
- 工具列: ![](../../images/fontpopup.gif) (點擊箭頭)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FONT (4218)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4218);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4218
```
