# Markdown/HTML 工具列命令

## 摘要

顯示或隱藏 Markdown/HTML 工具列。

## 說明

顯示或隱藏 Markdown/HTML 工具列。

## 運行方法

- 預設功能表: **檢視** > **工具列** \> **Markdown/HTML**
- [全部命令](../tools/all_commands): **檢視** > **工具列** \> **Markdown/HTML**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_SHOW_MARKDOWN_BAR (23274)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23274);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23274
```
