# 字型命令

## 摘要

顯示快顯功能表來選擇一個字型。

## 說明

這個命令會顯示一個快顯功能表，其中包括 [**最近使用的字型** (多個項目) 命令](mru_font1)， **字型分類** 子功能表還有 [**字型** 命令](font)。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **字型** >
**字型** (快顯功能表)
- 工具列: ![](../../images/fontpopup..png) (點擊箭頭)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FONT_POPUP (4282)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4282);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4282
```
