# 將 Tab 轉換為空格命令

## 摘要

將 Tab 轉換為空格。

## 說明

當按下 Tab 鍵時，插入空格而不是一個 tab 字元。

## 運行方法

- 預設功能表: **檢視** \> **Tab/縮排** \> **將 Tab 轉換為空格**
- [全部命令](../tools/all_commands): **檢視** \> **Tab/縮排** \> **將 Tab 轉換為空格**
- 工具列:
![](../../images/space_tab24x16..png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_TOGGLE_SPACES_TAB (4541)
```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4541);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4541
```
