# 縮排欄命令

## 摘要

顯示一個快顯功能表來選擇縮排欄。

## 說明

顯示一個快顯功能表來選擇縮排欄。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **Tab/縮排** \> **縮排欄**
- Toolbar:
![](../../images/indent_column24x16..png)
- 狀態列: 無
- 預設捷徑: 無

```
EEID_INDENT_COLUMN_POPUP (4551)
```

editor.ExecuteCommandByID(4551);
