# 目前的文檔中的上一個書籤命令

## 摘要

轉到目前的文檔中的上一個書籤。

## 說明

把游標移動到目前的文檔中的上一個書籤處。如果在目前的文檔中不存在任何書籤的話，該命令將不會移動游標。

## 運行方法

- 預設功能表: **書籤** \> **上一個書籤**
- [全部命令](../tools/all_commands): **書籤** \> **上一個書籤**
- 工具列: ![](../../images/bookmarkprevwithin..png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_PREV_WITHIN (4352)
```

## 巨集

### \[JavaScript\]

```
document.selection.PreviousBookmark();
```

### \[VBScript\]

```
document.selection.PreviousBookmark
```
