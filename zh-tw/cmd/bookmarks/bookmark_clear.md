# 清除目前文件中的所有書籤命令

## 摘要

清除目前文件中的所有書籤。

## 說明

清除目前文件中的所有書籤。

## 運行方法

- 預設功能表: **書籤** \> **清除所有書籤**
- [全部命令](../tools/all_commands): **書籤** \> **清除所有書籤**
- 工具列: ![](../../images/bookmarkclear.png)
- 狀態列: 無
- 預設捷徑: CTRL+SHIFT+F2

## 外掛程式命令ID

```
EEID_BOOKMARK_CLEAR (4323)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4323);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4323
```
