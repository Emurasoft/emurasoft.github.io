# 選取帶標題的 CSV 欄命令

## 摘要

在 CSV 文檔中用垂直選擇模式選取目前的欄并包括標題。

## 說明

在 CSV 文檔中用垂直選擇模式選取目前的欄并包括標題。

## 運行方法

- 預設功能表: **CSV** \> **欄** \> **選取帶標題的欄**
- [全部命令](../tools/all_commands): **CSV** \> **欄** \> **選取帶標題的欄**
- 工具列: ![](../../images/columns_separators.png) \+ 選取帶標題的欄
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_SELECT_COLUMN_WITH_HEADINGS (3931)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3931);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3931
```
