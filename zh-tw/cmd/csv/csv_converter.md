# CSV 轉換器命令

## 摘要

顯示或隱藏 CSV 轉換器。

## 說明

顯示或隱藏 CSV 轉換器。

## 運行方法

- 預設功能表: **CSV** \> **CSV 轉換器**
- [全部命令](../tools/all_commands): **CSV** \> **CSV 轉換器**
- 工具列: ![](../../images/convert_to.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_CSV\_CONVERTER (3891)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3891);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3891
```
