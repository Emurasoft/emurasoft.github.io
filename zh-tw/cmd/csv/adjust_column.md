# 調整分隔符號位置命令 (整個文件) 命令

## 摘要

調整 CSV 文檔中所有分隔符號的位置。

## 說明

調整 CSV 文檔中所有分隔符號的位置。

## 運行方法

- 預設功能表: **CSV** \> **調整分隔符號位置**
- [全部命令](../tools/all_commands): **CSV** \> **調整分隔符號位置**
- 工具列: ![](../../images/columns_separators.gif) \+ 調整分隔符號位置
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_ADJUST_COLUMN (4460)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4460);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4460
```
