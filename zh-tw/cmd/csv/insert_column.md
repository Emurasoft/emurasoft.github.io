# 在左邊插入一欄命令

## 摘要

在含有分隔值的文檔中的目前的欄的左邊插入新的一欄。

## 說明

在含有分隔值的文檔中的目前的欄的左邊插入新的一欄。

## 運行方法

- 預設功能表: **CSV** \> **欄** \> **在左邊插入一欄**
- [全部命令](../tools/all_commands): **CSV** \> **欄** \> **在左邊插入一欄**
- 工具列: ![](../../images/columns_separators.png) \+ 在左邊插入一欄
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_INSERT\_COLUMN (3887)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3887);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3887
```
