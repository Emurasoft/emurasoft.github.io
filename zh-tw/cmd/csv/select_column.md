# 選取不帶標題的 CSV 欄命令

## 摘要

在 CSV 文檔中用垂直選擇模式選取目前的欄但不包括標題。

## 說明

在 CSV 文檔中用垂直選擇模式選取目前的欄但不包括標題。

## 運行方法

- 預設功能表: **CSV** \> **欄** \> **選取不帶標題的欄**
- [全部命令](../tools/all_commands): **CSV** \> **欄** \> **選取不帶標題的欄**
- 工具列: ![](../../images/columns_separators.gif) \+ 選取不帶標題的欄
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SELECT_COLUMN (4461)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4461);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4461
```
