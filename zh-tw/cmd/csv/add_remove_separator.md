# 添加/刪除分隔符號命令

## 摘要

在游標位置處(或在尺規上所指的位置處)添加或刪除一個分隔符號當在標準模式下準備轉換為固定欄寬模式時。

## 說明

在游標位置處(或在尺規上所指的位置處)添加或刪除一個分隔符號當在標準模式下準備轉換為固定欄寬模式時。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **CSV** \> **分隔符號** \> **添加/刪除分隔符號**
- 工具列: ![](../../images/columns_separators.png) \+ 添加/刪除分隔符號
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_ADD\_REMOVE\_SEPARATOR (3884)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3884);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3884
```
