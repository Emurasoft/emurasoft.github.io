# 添加欄命令

## 摘要

在含有分隔值的文檔中的最后一欄的右邊添加新的一欄。

## 說明

在含有分隔值的文檔中的最后一欄的右邊添加新的一欄。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **CSV** \> **欄** \> **添加欄**
- 工具列: ![](../../images/columns_separators.png) \+ 添加欄
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_ADD\_COLUMN (3888)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3888);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3888
```
