# 插入最近使用的檔案清單命令

## 摘要

插入一個指定的最近訪問的檔案在游標位置處 (多個項目) 。

## 說明

這個命令由多個功能表項目組成，讓您能選擇一個最近訪問的檔案在游標位置處插入。所顯示的最近訪問的檔案數
可以在 **工具** 功能表 (**工具** \> **自訂** \> **歷史記錄**) 下的 [**自訂** 對話方塊](../../dlg/customize/index) 中 [**歷史記錄** 頁面](../../dlg/customize/history/index) 上的 **最近使用的檔案數** 文字方塊中設定。

## 運行方法

- 預設功能表: **插入** \> **(選擇檔案名)**
- [全部命令](../tools/all_commands): **插入** \> **(選擇檔案名)**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
從 EEID_FILE_MRU_INSERT1 到 EEID_FILE_MRU_INSERT1 + 63 (從 4864 到 4864 + 63)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4864 + i); // i 是一個從 0 到 63 的整數
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4864 + i   ' i 是一個從 0 到 63 的整數
```
