# 插入程式碼片段命令

## 摘要

在目前的游標位置插入指定的程式碼片段（多項）。

## 說明

該命令由多個功能表項組成，允許您選擇一個程式碼片段插入到游標位置。

## 運行方法

- 預設功能表：**插入** > **（插入程式碼片段）**
- [所有命令](../tools/all_commands)：**插入** > **程式碼片段** > **（插入程式碼片段）**
- 工具列：無
- 狀態列：無
- 預設快速鍵：無

## 外掛程式命令 ID

```
從 EEID_INSERT_SNIPPET1 到 EEID_INSERT_SNIPPET1 + 1023（從 30208 到 30208 + 1023）
```

## 巨集

## [JavaScript]

```
editor.ExecuteCommandByID(30208 + i); // i 是一個從 0 到 1023 的整數
```

## [VBScript]

```
editor.ExecuteCommandByID 30208 + i   ' i 是一個從 0 到 1023 的整數
```
