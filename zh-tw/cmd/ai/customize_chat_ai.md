# 自訂與 AI 聊天命令

## 摘要

打開自訂對話方塊中的與 AI 聊天頁面。

## 說明

顯示讓你能定義通用應用設定的[**自訂**對話方塊](../../dlg/customize/index)中的 **與 AI 聊天** 頁面。

## 運行方法

- 預設功能表：**AI** > **自訂** > **與 AI 聊天**
- [所有命令](../tools/all_commands)：**AI** > **自訂** > **與 AI 聊天**
- 工具列：無
- 狀態列：無
- 預設快速鍵：無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_CHAT_AI (9074)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9074);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9074
```
