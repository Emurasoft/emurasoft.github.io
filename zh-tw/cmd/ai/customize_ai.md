# 自訂 AI 選項命令

## 摘要

打開自訂對話方塊中的 AI 選項頁面。

## 說明

顯示讓你能定義通用應用設定的[**自訂**對話方塊](../../dlg/customize/index)中的 **AI 選項**頁面。

## 運行方法

- 預設功能表: **AI** > **自訂** > **AI 選項**
- [所有命令](../tools/all_commands): **AI** > **自訂** > **AI 選項**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_CUSTOMIZE_AI (9071)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9071);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9071
```
