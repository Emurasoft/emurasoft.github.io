# AI 工具列命令

## 摘要

顯示或隱藏 AI 工具列。

## 說明

顯示或隱藏 AI 工具列。

## 運行方法

- 預設功能表: **AI** \> **AI 工具列**
- [所有命令](../tools/all_commands): **AI** \> **AI 工具列**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_SHOW_AI_BAR (23253)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23253);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23253
```
