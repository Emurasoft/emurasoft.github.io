# 切換迷你地圖命令

## 摘要

切換迷你地圖。

## 說明

切換迷你地圖。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗
\>捲動
\>切換迷你地圖
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_MINIMAP\_TOGGLE (3975)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3975);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3975
```
