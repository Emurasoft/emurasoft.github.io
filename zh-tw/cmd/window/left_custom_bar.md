# 左側自訂欄命令

## 摘要

自訂欄居于左側。

## 說明

把自訂欄切換到左側如果可行的話。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **自訂欄** \> **左**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_FOCUS\_LEFT\_BAR (4425)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4425);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4425
```
