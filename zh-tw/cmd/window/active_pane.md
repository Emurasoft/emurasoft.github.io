# 活動窗格命令

## 摘要

切換到活動窗格。

## 說明

切換到活動窗格當顯示自訂欄時。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **窗格**
\> **活動窗格**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_ACTIVE\_PANE (4409)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4409);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4409
```
