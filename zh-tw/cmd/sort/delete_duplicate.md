# 刪除重複行命令

## 摘要

刪除已選中文字或整個文檔內的重複行。

## 說明

刪除已選中文字或整個文檔內的重複行。重複行第一個執行個體將被保留。

## 運行方法

- 預設功能表: **排序** \> **刪除重複行**
- [全部命令](../tools/all_commands): **排序** \> **刪除重複行**
- 工具列: ![](../../images/deleteduplicatelines..png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_DELETE_DUPLICATE (4564)
```

## 巨集

## \[JavaScript\]

```
document.DeleteDuplicates("",0);
```

## \[VBScript\]

```
document.DeleteDuplicates "",0
```
