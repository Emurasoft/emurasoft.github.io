# 按數字升序命令

## 摘要

在目前的欄中以升序排序數字。

## 說明

在目前的欄中以升序來排列數字。這個命令對非數字的字元不起作用。

## 運行方法

- 預設功能表: **排序** \> **按數值小到大排序**
- [全部命令](../tools/all_commands): **排序** \> **按數值小到大排序**
- 工具列: ![](../../images/sorting0-9..png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_NUM_A (4479)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4479);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4479
```
