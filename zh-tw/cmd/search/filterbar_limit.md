# 阻止對多行進行修改 (篩選工具列) 命令

## 摘要

切換「篩選」工具列上的「阻止對多行進行修改」按鈕。

## 說明

切換「篩選」工具列上的「阻止對多行進行修改」按鈕。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **搜索**
\> **篩選工具列** \> **阻止對多行進行修改**
- 工具列: ![](../../images/blockmultilinechanges.png) (篩選工具列)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_FILTERBAR\_LIMIT(3915)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3915);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3915
```
