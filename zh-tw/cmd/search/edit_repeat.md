# 尋找下一個命令

## 摘要

尋找下一個符合。

## 說明

用同樣的參數尋找與之前搜尋過的字串相同的下一個符合。

## 運行方法

- 預設功能表: **搜尋** \> **下一個**
- [全部命令](../tools/all_commands): **搜尋**
\> **下一個**
- 工具列: ![](../../images/editrepeat..png)
- 狀態列: 無
- 預設捷徑: F3

## 外掛程式命令ID

```
EEID_EDIT_REPEAT (4202)
```

## 巨集

### \[JavaScript\]

```
document.selection.FindRepeat(eeFindRepeatNext);
```

### \[VBScript\]

```
document.selection.FindRepeat eeFindRepeatNext
```
