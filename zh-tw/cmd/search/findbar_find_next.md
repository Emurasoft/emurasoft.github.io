# 尋找下一個 (搜尋工具列) 命令

## 摘要

用搜尋工具列尋找下一個符合。

## 說明

用搜尋工具列尋找下一個符合。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):搜尋
\>搜尋工具列 \>尋找下一個
- 工具列: ![](../../images/editrepeat.gif) (搜尋工具列)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FINDBAR_FIND_NEXT (4570)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4570);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4570
```
