# 尋找命令

## 摘要

尋找一個字串。

## 說明

尋找一個指定的字串。這個命令顯示 [尋找對話方塊](../../dlg/find/index)。指定一個字串和其他參數會，就可以開始多檔尋找指定的字串。

## 運行方法

- 預設功能表: **搜尋** \> **尋找**
- [全部命令](../tools/all_commands): **搜尋**
\> **尋找**
- 工具列: ![](../../images/editfind..png)
- 狀態列: 無
- 預設捷徑: CTRL+F

## 外掛程式命令ID

```
EEID_EDIT_FIND (4200)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4200);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4200
```
