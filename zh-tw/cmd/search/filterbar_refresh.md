# 重新整理命令

## 摘要

用目前的篩選設定重新整理文檔。

## 說明

用目前的篩選設定重新整理文檔。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **搜索**
\> **篩選工具列** \> **重新整理**
- 工具列: ![](../../images/refresh..png) (篩選工具列)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_FILTERBAR\_REFRESH (3905)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3905);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3905
```
