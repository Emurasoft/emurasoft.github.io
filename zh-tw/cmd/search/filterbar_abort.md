# 中止 (篩選工具列) 命令

## 摘要

中止篩選并清空篩選字串。

## 說明

中止篩選并清空篩選字串。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **搜索**
\> **篩選工具列** \> **中止**
- 工具列: ![](../../images/abort.png) (篩選工具列)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_FILTERBAR\_ABORT (3914)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3914);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3914
```
