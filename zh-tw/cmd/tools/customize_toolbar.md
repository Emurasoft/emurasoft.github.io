# 自訂工具列命令

## 摘要

自訂工具列設置。

## 說明

顯示**自訂**對話方塊中的[**工具列按鈕**](../../dlg/customize/toolbar_buttons/index)頁面，您可以在此自訂工具列。

## 運行方法

- 預設功能表: **工具** \> **自訂工具列**
- [所有命令](all_commands): **工具** >
**自訂工具列**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_TOOLBAR (4224)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4224);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4224
```
