# 新提示命令

## 摘要

輸入一個新提示來詢問 AI。

## 說明

顯示[**新提示**對話方塊](../../dlg/new_prompt/index)以建立一個新的提示。

## 運行方法

- 預設功能表: **工具** \> **AI** \> **新提示**
- [所有命令](all_commands): **AI** \> **新提示**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_NEW_PROMPT (23252)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23252);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23252
```
