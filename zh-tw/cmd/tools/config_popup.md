# 選擇組態 (快顯功能表) 命令

## 摘要

選擇一個組態。

## 說明

用一個快顯功能表來顯示 [**組態清單** 命令](select_config) (多個項目) 還有 [**定義組態** 命令](config)。

## 運行方法

- 預設功能表: **工具** >
**選擇組態** \> **選擇組態 (快顯功能表)**
- [所有命令](all_commands): **工具** >
**選擇組態** \> **選擇組態 (快顯功能表)**
- 工具列: ![](../../images/configpopup.png) (在箭頭上)
- 狀態列: (雙擊組態名稱)
- 預設捷徑: SHIFT + F11

## 外掛程式命令ID

```
EEID_CONFIG_POPUP (4220)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4220);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4220
```
