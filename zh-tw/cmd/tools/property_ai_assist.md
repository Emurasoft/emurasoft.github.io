# AI 助理屬性命令

## 摘要

目前的組態的 AI 助理屬性。

## 說明

顯示 **[組態屬性](../../dlg/properties/index)** 對話方塊中的 [**AI 助理** 頁面](../../dlg/properties/ai_assist/index)。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** > **當前組態屬性** \> **AI 助理**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_PROPERTY_AI_ASSIST (8983)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(8983);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8983
```
