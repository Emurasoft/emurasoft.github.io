# 列印命令

## 摘要

列印目前的檔案。

## 說明

這個命令顯示 **列印** 對話方塊，讓您能指定要復印的數目以及文檔列印的範圍。點擊 **OK** 來開始列印。

## 運行方法

- 預設功能表: **檔案** \> **列印**
- [全部命令](../tools/all_commands): **檔案** \> **列印** \> **列印**
- 工具列: ![](../../images/fileprint.gif)
- 狀態列: 無
- 預設捷徑: CTRL+P
