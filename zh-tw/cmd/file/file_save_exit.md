# 儲存并關閉命令

### 摘要

> 儲存并關閉目前的檔案。

### 說明

> 把變更儲存為，并關閉目前的文檔。

### 運行方法

- 預設功能表: **檔案** \> **儲存并關閉**
- [全部命令](../tools/all_commands): **檔案** \> **關閉**
\> **儲存并關閉**
- 工具列: ![](../../images/filesaveexit.gif)
- 狀態列: 無
- 預設捷徑: CTRL+E

### 外掛程式命令ID

- EEID\_FILE\_SAVE\_EXIT (4116)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4116);

#### \[VBScript\]

> editor.ExecuteCommandByID 4116
