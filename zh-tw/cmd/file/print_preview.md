# 列印預覽命令

## 摘要

顯示目前的文檔的列印預覽。

## 說明

顯示 **列印預覽** 視窗預覽目前的文檔的列印效果。在 **列印預覽** 視窗中，您可以列印或設置列印參數。

## 運行方法

- 預設功能表: **檔案** \> **列印預覽**
- [全部命令](../tools/all_commands): **檔案** \> **列印**
\> **列印預覽**
- 工具列: ![](../../images/printpreview.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_PRINT_PREVIEW (4286)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4286);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4286
```
